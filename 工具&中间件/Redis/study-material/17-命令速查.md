# Redis学习资料：命令速查

[返回索引](../Redis学习资料.md)

## 连接和状态

```bash
redis-cli -h 127.0.0.1 -p 6379
PING
INFO memory
INFO replication
CLIENT LIST
SLOWLOG GET 10
```

## Key 和 TTL

```bash
EXISTS key
TYPE key
TTL key
EXPIRE key 3600
PERSIST key
DEL key
UNLINK key
SCAN 0 MATCH 'user:*' COUNT 1000
```

## String

```bash
SET key value EX 60 NX
GET key
MGET k1 k2
INCR counter
DECR counter
SETNX lock value
```

## Hash

```bash
HSET user:1 name alice age 20
HGET user:1 name
HMGET user:1 name age
HINCRBY user:1 count 1
HSCAN user:1 0
```

## List

```bash
LPUSH queue job
RPOP queue
BRPOP queue 5
LRANGE list 0 9
LTRIM list 0 99
```

## Set

```bash
SADD set a b
SISMEMBER set a
SCARD set
SINTER set1 set2
SSCAN set 0
```

## ZSet

```bash
ZADD rank 100 user1
ZREVRANGE rank 0 9 WITHSCORES
ZRANGEBYSCORE rank 0 100
ZREM rank user1
ZSCAN rank 0
```

## Stream

```bash
XADD s * k v
XREAD COUNT 10 STREAMS s 0
XGROUP CREATE s g $ MKSTREAM
XREADGROUP GROUP g c COUNT 10 STREAMS s >
XACK s g id
XPENDING s g
```

## 事务和 Lua

```bash
MULTI
SET a 1
INCR b
EXEC

EVAL "return redis.call('GET', KEYS[1])" 1 key
```

## 高风险命令

```bash
KEYS *
MONITOR
FLUSHALL
FLUSHDB
CONFIG SET
DEBUG
```

> **易错：** 速查不等于生产可直接执行。高风险命令必须确认影响范围。
