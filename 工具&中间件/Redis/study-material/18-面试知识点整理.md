# Redis学习资料：面试知识点整理

[返回索引](../Redis学习资料.md)

这份资料用于 Redis 面试前复习。面试问题按知识点拆分到 `面试知识点/` 目录。

## 使用方式

- 初级：先看基础概念、数据结构和缓存场景。
- 后端：重点看缓存一致性、分布式锁、Lua、性能和高可用。
- 中高级：重点看事件循环、对象编码、持久化、复制、Cluster 和生产故障。
