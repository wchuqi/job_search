# Redis学习资料：深度实验手册和能力验收

[返回索引](../Redis学习资料.md)

## 实验 1：数据类型和编码

```bash
HSET h a 1 b 2
OBJECT ENCODING h
MEMORY USAGE h
```

不断增加字段，观察编码和内存变化。

## 实验 2：过期和淘汰

- 创建带 TTL key。
- 设置 maxmemory 和 allkeys-lru。
- 写入大量 key。
- 观察 `expired_keys`、`evicted_keys`。

## 实验 3：big key 阻塞

- 创建大 Set。
- 执行 `SMEMBERS`。
- 同时从另一个客户端执行 `PING`。
- 观察延迟。

## 实验 4：Lua 原子锁释放

- SET NX PX 加锁。
- Lua 校验 value 删除。
- 模拟锁过期后其他客户端拿到锁，验证不能误删。

## 实验 5：AOF rewrite 和 fork

- 开启 AOF。
- 大量写入。
- 触发 BGREWRITEAOF。
- 观察 `INFO persistence` 和内存。

## 实验 6：主从复制

- 搭建主从。
- 观察 offset。
- 暂停从节点网络。
- 恢复后观察部分同步或全量同步。

## 实验 7：Cluster hash tag

- 创建 Cluster。
- 用 `CLUSTER KEYSLOT` 比较 key。
- 设计同槽多 key。
- 模拟 MOVED/ASK。

## 实验 8：Stream 消费组

- XADD 消息。
- XREADGROUP 消费。
- 不 ACK，观察 XPENDING。
- 用 XAUTOCLAIM 重新领取。

## 最终能力验收

- 能解释 Redis 单线程模型和阻塞来源。
- 能解释对象编码和内存开销。
- 能设计缓存一致性和热点保护。
- 能搭建主从、Sentinel 或 Cluster 实验。
- 能定位 slowlog、big key、hot key、复制延迟。
- 能写 Redis 生产配置、安全和备份恢复清单。
