# Nginx学习资料：容量压测、连接模型、队列和成本治理深度解析

[返回索引](../Nginx学习资料.md)

## 学习目标

- 建立 Nginx 容量压测模型。
- 计算连接、带宽、TLS、upstream 和日志成本。
- 设计 SLA 和扩容阈值。

## 容量维度

```text
峰值 QPS
并发连接
长连接数量
平均响应大小
P95/P99 延迟
TLS 握手率
upstream 连接数
日志写入量
缓存命中率
```

## 压测注意

- 使用真实 Host、TLS、header。
- 模拟 keepalive。
- 区分静态文件和代理。
- 包含慢客户端。
- 包含后端超时和失败。
- 观察 Nginx、upstream、内核指标。

## 成本

- TLS CPU。
- 日志磁盘。
- proxy temp 磁盘。
- 缓存磁盘。
- 网络带宽。
- upstream 连接和应用线程。

## 验收

- 能写容量表。
- 能设计压测场景。
- 能定义扩容和告警阈值。

## 易错

> **易错：** 只用短连接 HTTP 压测代表生产。
>
> 正确做法：模拟真实协议、keepalive、TLS、响应大小、慢客户端和 upstream 行为。
