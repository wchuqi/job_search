# Nginx学习资料：综合练习项目

[返回索引](../Nginx学习资料.md)

## 项目目标

构建一个生产风格 Nginx 入口：HTTPS、前端静态站点、API 反向代理、WebSocket、缓存、限流、日志和灰度。

## 结构

```text
nginx-lab/
  nginx.conf
  conf.d/
    app.conf
  certs/
  html/
```

## 功能

- HTTP 跳 HTTPS。
- `/` 服务 SPA。
- `/assets/` 长缓存。
- `/api/` 代理到后端。
- `/ws/` WebSocket。
- `/cache/` proxy_cache。
- `/admin/` IP 白名单。
- access_log 包含 upstream 时间。

## 验收

- `nginx -t` 通过。
- curl 验证 Host、HTTPS、缓存头。
- 后端关闭时返回可解释 502。
- 限流能触发。
- 日志能定位 upstream。

## 重点

- 项目不是堆配置，而是验证每条配置的行为。
- 所有功能都要有 curl 验证命令。
