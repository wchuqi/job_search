# Maven 学习资料：命令速查

[返回索引](../Maven学习资料.md)

## 使用原则

速查命令前先判断问题属于哪一类：生命周期执行、依赖分析、POM 查看、多模块构建、测试控制、仓库发布，还是排障调试。

## 生命周期命令

```bash
mvn clean
mvn validate
mvn compile
mvn test
mvn package
mvn verify
mvn install
mvn deploy
```

## 常用组合

```bash
mvn clean package
mvn clean verify
mvn -B clean verify
mvn clean install -DskipTests
```

## 依赖分析

```bash
mvn dependency:tree
mvn dependency:tree -Dincludes=org.slf4j
mvn dependency:analyze
mvn dependency:resolve
```

## POM 和 Profile

```bash
mvn help:effective-pom
mvn help:effective-settings
mvn help:active-profiles
mvn help:describe -Dplugin=compiler
```

## 多模块

```bash
mvn -pl module-a test
mvn -pl module-a -am test
mvn -pl module-a -amd test
mvn -rf :module-a package
```

## 测试控制

```bash
mvn test
mvn test -Dtest=UserServiceTest
mvn package -DskipTests
mvn package -Dmaven.test.skip=true
```

## 排障

```bash
mvn -X package
mvn -e package
mvn -U clean package
mvn -o test
```

## 发布

```bash
mvn install
mvn deploy
```

## 重点

> **重点：** `install` 到本地仓库，`deploy` 到远程仓库。

## 易错

> **易错：** `-DskipTests` 和 `-Dmaven.test.skip=true` 不同；前者跳过运行测试，后者连测试编译也跳过。
