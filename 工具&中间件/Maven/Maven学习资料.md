# Maven 学习资料

这份资料以 Maven 3.x 的主流使用方式为基础，面向 Java 后端和工程化学习。内容按知识点拆分到 `study-material/` 目录，覆盖 Maven 心智模型、POM、依赖管理、生命周期、插件、多模块、发布部署、CI、排障、安全和面试复习。

## 阅读原则：先理解模型，再背命令

Maven 不是一个只会执行 `mvn package` 的打包工具。它更像一套“项目描述 + 依赖解析 + 生命周期编排 + 插件执行”的构建系统：`pom.xml` 描述项目，坐标定位构件，仓库保存构件，生命周期规定构建阶段，插件真正执行编译、测试、打包、部署等动作。

学习 Maven 时，每个知识点都要能回答：它解决什么问题、它在构建链路中处于什么位置、它影响依赖还是构建行为、它的默认规则是什么、误用会造成什么后果。命令只是触发构建的入口，真正要理解的是 Maven 如何根据 POM 和生命周期把项目一步步构建出来。

## 使用建议

- 初学按 0 到 4 学习 Maven 基础和依赖模型。
- 做企业 Java 项目，重点学习 4、5、6、7、8。
- 排查构建失败，优先查 3、4、9、11。
- 面试前先看 12 和 13，再回到 2、3、4、5 复习核心机制。
- 每学一个命令，都要说清它触发哪个生命周期阶段、哪些插件目标会运行、依赖如何解析。
