## 高频考点

- REST 和 RPC 的区别。
- HTTP 方法的安全性和幂等性。
- `PUT` 和 `PATCH` 的区别。
- `401` 和 `403` 的区别。
- `400`、`409`、`422` 的区别。
- 分页、过滤、排序怎么设计。
- 如何防止重复提交。
- API 版本如何演进。
- 如何做鉴权、限流、日志和链路追踪。
- HATEOAS 为什么工程中少见。
- 复杂动作如何建模为资源、任务或命令。
- 幂等键、唯一约束、状态机如何配合。
- OAuth2、OIDC、JWT、CORS、CSRF 的边界。
- 如何识别 API 破坏性变更。
