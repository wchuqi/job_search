# 分布式锁与事务完整知识点清单

[返回索引](../分布式锁与事务学习资料.md)

## 1. 基础和术语

- ACID、BASE、CAP、最终一致、强一致、线性一致。
- 事务、锁、租约、会话、幂等、补偿、对账。
- 临界区、业务不变量、聚合根、状态机。
- 至少一次、至多一次、恰好一次的工程含义。

## 2. 本地事务和并发控制

- 事务隔离级别。
- 脏读、不可重复读、幻读、丢失更新、写偏斜。
- 行锁、间隙锁、表锁、死锁。
- 乐观锁、版本号、CAS。
- 唯一约束、外键、检查约束。
- 状态机条件更新。

## 3. 分布式锁

- 锁 key 设计、锁粒度、锁超时。
- requestId、owner、TTL、续期、释放校验。
- 安全性、活性、公平性、可重入、可续期、可恢复性。
- 租约有效期和本地时钟、服务端时钟、网络延迟、进程暂停之间的关系。
- Redis `SET NX PX`、Lua 释放、主从切换风险。
- Redlock 算法、适用边界和争议。
- fencing token、单调序号、下游拒绝旧请求。
- fencing token 与乐观锁 version、etcd revision、ZK zxid/epoch 的关系。
- ZooKeeper 临时节点、临时顺序节点、watch、会话。
- etcd lease、revision、watch、leader election。
- 锁竞争、惊群、热点 key、锁泄漏、脑裂。
- 正确性证明：旧持有者恢复、双主、续期失败、锁服务 failover、下游乱序写入。

## 4. 分布式事务

- 2PC prepare/commit。
- 2PC 阻塞、协调器故障、参与者故障。
- 3PC 的阶段和现实局限。
- XA 事务管理器和资源管理器。
- 事务日志、恢复、启发式提交。
- 长事务和资源锁定风险。
- in-doubt 事务、prepared transaction、事务管理器重启恢复。
- 参与者 prepare 成功后本地资源锁定和 undo/redo 语义。
- 二阶段部分 commit 后的恢复、查询事务结果和人工决议。

## 5. 最终一致方案

- Saga 编排式和协同式。
- Saga 状态机、补偿、重试、超时。
- Saga 实例表、步骤表、补偿日志、超时扫描、并发推进控制。
- TCC Try/Confirm/Cancel。
- TCC 空回滚、悬挂、幂等。
- TCC 分支事务表、全局事务状态、反悬挂机制、冻结资源模型。
- 事务消息、半消息、事务回查。
- Outbox、Inbox、CDC 发布。
- Outbox 扫描并发控制、CDC 位点、消息键分区、事件版本和顺序。
- 死信队列、延迟队列、重试退避。

## 6. 幂等和去重

- API 幂等键。
- 消息 ID 去重。
- 业务流水号。
- 请求摘要校验。
- 状态机幂等。
- 数据库唯一索引幂等。
- Redis 去重的过期风险。
- 幂等记录保留周期和审计。
- 幂等键请求摘要校验、防重放、跨租户隔离。
- 乱序事件处理、事件版本、聚合根串行化、最终状态优先级。

## 7. 对账和补偿

- 内部系统对账。
- 外部支付渠道对账。
- 补偿任务、重试策略、退避。
- 自动修复和人工处理边界。
- 审计日志、操作人、原因码。
- SLA、超时扫描、卡单处理。

## 8. 生产和运维

- 锁获取成功率、等待时间、持锁时间。
- 锁续期失败、释放失败、TTL 过期次数。
- Saga 卡住数量、补偿失败数量。
- Outbox 积压、发布失败、死信数量。
- 消费延迟、重复消费、乱序事件。
- traceId、业务流水号、聚合根 ID。
- 告警、仪表盘、运行手册、故障演练。

## 9. 安全和合规

- 幂等键绑定用户和请求摘要。
- 补偿后台权限和审批。
- 支付、退款、权益操作审计。
- 日志脱敏。
- 防止恶意重放请求。

## 10. 面试高频判断点

- 是否能先谈业务不变量，而不是直接套 Redis 锁。
- 是否知道锁过期后旧客户端继续执行的问题。
- 是否能说出 fencing token 必须由下游校验。
- 是否理解 2PC 阻塞和 XA 边界。
- 是否能解释 Saga/TCC 的失败路径。
- 是否知道消息消费者必须幂等。
- 是否能设计对账和人工兜底。
