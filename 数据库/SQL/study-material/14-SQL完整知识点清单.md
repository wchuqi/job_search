# SQL 完整知识点清单

[返回索引](../SQL学习资料.md)

## 一、基础和关系模型

- SQL、关系模型、表、行、列、关系、结果集。
- 主键、候选键、外键、唯一约束、检查约束、默认值。
- 声明式语言和过程式语言差异。
- 逻辑执行顺序和物理执行计划差异。
- 多重集合语义：SQL 结果默认允许重复行。

## 二、基础查询

- `SELECT`、列别名、表达式。
- `FROM`、表别名。
- `WHERE`、比较、范围、集合、模糊匹配。
- `ORDER BY`、多列排序、NULL 排序差异。
- `LIMIT/OFFSET`、`FETCH FIRST`、TopN。
- Keyset pagination。
- 日期范围半开区间。

## 三、NULL 和表达式

- NULL 含义和三值逻辑。
- TRUE、FALSE、UNKNOWN。
- `IS NULL`、`IS NOT NULL`。
- `COALESCE`、`NULLIF`、`CASE WHEN`。
- `COUNT(*)` 和 `COUNT(col)`。
- `NOT IN` 与 NULL。
- NULL 在唯一约束和排序中的方言差异。

## 四、JOIN 和关系运算

- INNER JOIN、LEFT JOIN、RIGHT JOIN、FULL OUTER JOIN。
- CROSS JOIN。
- 自连接。
- 半连接：`EXISTS`。
- 反连接：`NOT EXISTS`、`LEFT JOIN ... IS NULL`。
- 连接基数：一对一、一对多、多对多。
- 外连接 `ON` 和 `WHERE` 过滤差异。
- JOIN 后聚合行数放大。

## 五、聚合和去重

- `GROUP BY` 粒度。
- 聚合函数：`COUNT`、`SUM`、`AVG`、`MIN`、`MAX`。
- `HAVING`。
- 条件聚合：`SUM(CASE WHEN ...)`、`FILTER`。
- `DISTINCT`。
- 分组集合：`ROLLUP`、`CUBE`、`GROUPING SETS`，按数据库支持学习。

## 六、子查询、派生表和 CTE

- 标量子查询。
- 行子查询、表子查询。
- 相关子查询。
- `IN`、`EXISTS`、`ANY`、`ALL`。
- 派生表。
- CTE。
- 递归 CTE。
- CTE 物化和内联差异。

## 七、窗口函数

- `OVER`。
- `PARTITION BY`。
- 窗口内 `ORDER BY`。
- 窗口帧：`ROWS`、`RANGE`、`GROUPS`。
- `ROW_NUMBER`、`RANK`、`DENSE_RANK`。
- `LAG`、`LEAD`。
- `FIRST_VALUE`、`LAST_VALUE`。
- 累计和、移动平均、分组 TopN。

## 八、DML

- `INSERT`。
- 多行插入。
- `UPDATE`。
- `DELETE`。
- `TRUNCATE`，方言和事务语义需单独确认。
- UPSERT：PostgreSQL `ON CONFLICT`、MySQL `ON DUPLICATE KEY UPDATE`、SQL Server/Oracle `MERGE`。
- 批量写入。
- 条件更新和乐观锁。

## 九、DDL 和建模

- `CREATE TABLE`、`ALTER TABLE`、`DROP TABLE`。
- 数据类型：整数、小数、字符串、日期时间、布尔、JSON。
- 主键设计：自然键、代理键。
- 外键和级联动作。
- 范式：1NF、2NF、3NF、BCNF。
- 反范式、汇总表、宽表。
- 审计字段、软删除、状态机。

## 十、事务和并发

- ACID。
- 事务边界。
- 隔离级别。
- 脏读、不可重复读、幻读。
- 丢失更新、写偏斜。
- 锁等待、死锁。
- MVCC 概念，不同数据库实现不同。
- 悲观锁、乐观锁。
- 幂等写入。

## 十一、索引和执行计划

- B+Tree 索引。
- 哈希索引、位图索引、全文索引、GIN/GiST 等按数据库学习。
- 单列索引、组合索引。
- 覆盖索引。
- 选择性和基数。
- SARGability。
- 隐式类型转换。
- 函数包列。
- 前导通配 LIKE。
- 执行计划：扫描、连接、排序、聚合、回表。
- Nested Loop、Hash Join、Merge Join。
- 统计信息和基数估算。

## 十二、性能优化

- 慢查询采集。
- 执行计划分析。
- 谓词下推。
- 投影裁剪。
- JOIN 顺序和连接算法。
- TopN 和排序优化。
- 深分页优化。
- 聚合预计算。
- 分区裁剪。
- 参数敏感计划。
- 读写分离和报表异步化。

## 十三、安全和权限

- SQL 注入。
- 参数化查询。
- 动态 SQL 白名单。
- 最小权限。
- 行级权限，按数据库支持学习。
- 敏感字段脱敏。
- 审计日志。
- 备份、导出和报表权限。

## 十四、方言和迁移

- 分页语法差异。
- 字符串拼接差异。
- 日期函数差异。
- UPSERT/MERGE 差异。
- 布尔类型差异。
- JSON 函数差异。
- 标识符大小写和引号。
- NULL 排序差异。
- 事务隔离实现差异。
- CTE 优化策略差异。

## 十五、必须能回答的场景

- 为什么 `NOT IN` 遇到 NULL 会出问题。
- LEFT JOIN 后 WHERE 右表条件为什么会丢行。
- GROUP BY 后为什么不能随便 SELECT 明细列。
- 窗口函数和聚合函数有什么区别。
- 每组最新一条记录怎么写。
- 如何防止重复投递。
- 慢查询如何排查。
- 组合索引列顺序如何设计。
- 深分页如何优化。
- SQL 注入如何防。

- 统计信息、直方图、最常见值、NULL 比例、列相关性。
- 基数估算错误如何传递到 JOIN 顺序和 JOIN 算法。
- B+Tree 叶子页、有序扫描、范围扫描、覆盖索引、回表。
- 组合索引中的等值列、范围列、排序列和覆盖列取舍。
- 索引写入成本、冗余索引、索引膨胀和缓存成本。
- 锁、MVCC、快照读、当前读、写写冲突。
- 丢失更新、写偏斜、幻读在不同数据库中的表现。
- 递归 CTE 的环检测、深度限制和路径构造。
- 复杂报表的事实粒度、维度口径、迟到数据和质量校验。
