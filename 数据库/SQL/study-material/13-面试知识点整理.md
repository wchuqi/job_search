# SQL 学习资料：面试知识点整理

[返回索引](../SQL学习资料.md)

这份资料用于 SQL 面试前复习。面试问题已按知识点拆分到 `面试知识点/` 目录，覆盖查询基础、JOIN、聚合窗口、事务索引、性能优化、方言安全和高阶场景。

## 使用方式

- 初级：先看 01、02，确保基础查询和 JOIN 不犯错。
- 中级：重点看 03、04，掌握聚合窗口、事务索引和执行计划。
- 高级：重点看 05、06，准备慢查询、迁移、安全和深度追问。
- 回答原则：先给结论，再解释语义规则，最后给 SQL 示例、边界和性能影响。
