## 使用建议

- 零基础或长期复习：按 0 到 15 顺序阅读。
- 面试冲刺：优先看 11、12、13、14、15。
- 刷题卡壳：先看题型对应章节，再看 12 的题解表达方式。
- Java 后端岗位：重点掌握 Java 21 模板、集合 API、堆、队列、图和 DP。
- Python 快速刷题：重点掌握 `dict`、`set`、`deque`、`heapq`、排序 key 和递归限制。

> **重点：** 题解要能口述“题目特征、算法选择、状态含义、边界条件、复杂度”，只给代码不算真正掌握。
