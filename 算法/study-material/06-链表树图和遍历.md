## 理论导读

链表、树和图都不是连续下标结构，而是通过引用或边连接节点。处理这类题时，最重要的是明确“当前节点是谁、下一步去哪里、已经访问过什么、函数返回什么”。

树是没有环的特殊图。树题适合用递归，因为每个节点的子问题结构相同；图题通常必须显式记录 visited，否则会重复访问甚至死循环。

## Java 21 示例：二叉树层序遍历

```java
import java.util.*;

class TreeNode {
    int val;
    TreeNode left;
    TreeNode right;
    TreeNode(int val) { this.val = val; }
}

class Solution {
    public List<List<Integer>> levelOrder(TreeNode root) {
        List<List<Integer>> ans = new ArrayList<>();
        if (root == null) return ans;
        Queue<TreeNode> q = new ArrayDeque<>();
        q.offer(root);
        while (!q.isEmpty()) {
            int size = q.size();
            List<Integer> level = new ArrayList<>();
            for (int i = 0; i < size; i++) {
                TreeNode node = q.poll();
                level.add(node.val);
                if (node.left != null) q.offer(node.left);
                if (node.right != null) q.offer(node.right);
            }
            ans.add(level);
        }
        return ans;
    }
}
```

## Python 3 示例：二叉树层序遍历

```python
from collections import deque
from typing import Optional

class TreeNode:
    def __init__(self, val: int = 0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right

class Solution:
    def levelOrder(self, root: Optional[TreeNode]) -> list[list[int]]:
        if root is None:
            return []
        ans: list[list[int]] = []
        q = deque([root])
        while q:
            level = []
            for _ in range(len(q)):
                node = q.popleft()
                level.append(node.val)
                if node.left:
                    q.append(node.left)
                if node.right:
                    q.append(node.right)
            ans.append(level)
        return ans
```

## 验收

- 能用递归和迭代各写一种树遍历。
- 能说明图 BFS 中队列保存的是什么。

> **重点：** 链表题修改指针前先保存后继节点。
>
> **难点：** 树的递归题必须先定义函数含义，再写左右子树递归。
>
> **易错：** 图题忘记 visited，导致重复访问或无限递归。
