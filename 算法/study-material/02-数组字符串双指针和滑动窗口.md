- 掌握数组和字符串的线性扫描技巧。

## 理论导读

双指针的核心是用两个位置表达一个范围、关系或进度。它把原本需要枚举所有组合的问题，压缩成“每个元素最多进出一次”的线性过程。滑动窗口是双指针的一种特例，窗口中维护当前子数组或子串的状态。

窗口像一段可伸缩的尺子：右端负责纳入新元素，左端负责移除不再合法的元素。关键不是移动指针，而是定义“窗口内状态”和“合法条件”。

## 模型对比

| 模型 | 特征 | 典型题 |
| --- | --- | --- |
| 左右指针 | 数组通常已排序，从两端夹逼 | 两数之和 II、三数之和、盛最多水 |
| 快慢指针 | 一个指针探索，一个指针维护结果 | 删除重复项、链表找环 |
| 固定窗口 | 窗口长度固定 | 长度为 K 的最大平均值 |
| 可变窗口 | 窗口长度随条件变化 | 无重复最长子串、最小覆盖子串 |

def sliding_window(nums):
    left = 0
    state = {}
    ans = 0
    for right, x in enumerate(nums):
        # 加入 nums[right]
        while left <= right and window_invalid(state):
            # 移除 nums[left]
            left += 1

## Java 21 示例：无重复字符的最长子串

```java
import java.util.*;

class Solution {
    public int lengthOfLongestSubstring(String s) {
        Map<Character, Integer> last = new HashMap<>();
        int left = 0, ans = 0;
        for (int right = 0; right < s.length(); right++) {
            char c = s.charAt(right);
            if (last.containsKey(c)) {
                left = Math.max(left, last.get(c) + 1);
            }
            last.put(c, right);
            ans = Math.max(ans, right - left + 1);
        }
        return ans;
    }
}
```

## Python 3 示例：无重复字符的最长子串

```python
class Solution:
    def lengthOfLongestSubstring(self, s: str) -> int:
        last: dict[str, int] = {}
        left = ans = 0
        for right, ch in enumerate(s):
            if ch in last:
                left = max(left, last[ch] + 1)
            last[ch] = right
            ans = max(ans, right - left + 1)
        return ans
```

## 验收

- 能说明窗口内维护了哪些变量。
- 能解释为什么每个元素最多被加入和移除一次。

> **重点：** 滑动窗口要求窗口状态可以随着左右端移动高效更新。
>
> **难点：** 有负数的“子数组和”问题通常不能直接使用普通滑动窗口。
>
> **易错：** 遇到重复字符时把 `left` 往回移动。正确做法是 `left = max(left, last[ch] + 1)`。
