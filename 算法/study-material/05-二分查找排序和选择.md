# 算法学习资料：二分查找、排序和选择算法

[返回索引](../算法学习资料.md)

## 学习目标

- 掌握二分查找的边界写法。
- 理解排序算法和稳定性。
- 掌握第 K 大、第 K 小等选择问题。

## 理论导读

二分不是只在数组里找某个数，而是在有序答案空间中排除一半不可能区域。只要能定义一个单调判断函数，就可能二分答案。

排序不仅用于排列数据，也经常用于创造有序性，让双指针、贪心、去重更容易进行。选择算法则是在不完全排序的情况下找到排名元素，常见方案是堆或快速选择。

## 二分模板：左闭右闭

```java
int left = 0, right = nums.length - 1;
while (left <= right) {
    int mid = left + (right - left) / 2;
    if (nums[mid] == target) return mid;
    if (nums[mid] < target) left = mid + 1;
    else right = mid - 1;
}
return -1;
```

## 二分答案模型

```python
def binary_answer(left: int, right: int) -> int:
    while left < right:
        mid = (left + right) // 2
        if check(mid):
            right = mid
        else:
            left = mid + 1
    return left
```

## 排序注意点

| 概念 | 含义 | 面试关注 |
| --- | --- | --- |
| 稳定排序 | 相等元素相对顺序不变 | 对象多关键字排序 |
| 原地排序 | 额外空间小 | 空间约束 |
| 比较排序下界 | 平均至少 `O(n log n)` | 理论边界 |
| 桶/计数排序 | 利用值域 | 值域小才适合 |

## 验收

- 能写出查找左边界和右边界。
- 能说明二分答案的 `check` 函数为什么单调。

> **重点：** 二分的关键是搜索区间定义和循环不变量。
>
> **难点：** 二分答案不是找数组元素，而是找满足条件的最小或最大答案。
>
> **易错：** `mid = (left + right) / 2` 在 Java 中可能溢出，推荐 `left + (right - left) / 2`。
