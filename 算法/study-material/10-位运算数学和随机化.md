## 理论导读

位运算把整数看成二进制集合。每一位可以表示某个状态是否存在，因此位掩码常用于小规模状态压缩。数学题常见陷阱是溢出、取模负数和边界。随机化题关注概率均匀性，而不是随便调用随机函数。

## 常用位运算

| 表达式 | 含义 |
| --- | --- |
| `x & 1` | 判断最低位是否为 1 |
| `x & (x - 1)` | 去掉最低位的 1 |
| `x & -x` | 取最低位的 1 |
| `x ^ x = 0` | 相同数异或抵消 |
| `mask | (1 << i)` | 加入第 i 个状态 |
| `mask & ~(1 << i)` | 移除第 i 个状态 |

## Java 21 示例：只出现一次的数字

```java
class Solution {
    public int singleNumber(int[] nums) {
        int ans = 0;
        for (int x : nums) {
            ans ^= x;
        }
        return ans;
    }
}
```

## Python 3 示例：最大公约数

```python
def gcd(a: int, b: int) -> int:
    while b:
        a, b = b, a % b
    return abs(a)
```

## 随机化注意点

- 洗牌算法使用 Fisher-Yates，从后往前随机交换。
- 随机集合需要数组支持随机下标，哈希表支持 `O(1)` 定位。
- 权重随机通常用前缀和加二分。
