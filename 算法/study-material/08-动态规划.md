# 算法学习资料：动态规划

## 学习目标

- 掌握 DP 的状态定义、转移方程、初始化和遍历顺序。
- 能识别常见 DP 类型。
- 能用 Java 21 和 Python 3 写出基础 DP。

## 理论导读

动态规划适合有重叠子问题和最优子结构的问题。它像一本账本：每个状态的答案只计算一次，后面的状态直接引用已有结果。DP 最容易错的不是代码，而是状态定义不清。

写 DP 时必须按顺序回答四个问题：

1. `dp[i]` 或 `dp[i][j]` 表示什么。
2. 最后一步选择是什么。
3. 转移方程是什么。
4. 初始化和遍历顺序是什么。

## 类型速查

| 类型 | 状态例子 | 典型题 |
| --- | --- | --- |
| 一维 DP | `dp[i]` 前 i 个元素答案 | 爬楼梯、打家劫舍 |
| 二维 DP | `dp[i][j]` 两个维度状态 | 最长公共子序列 |
| 背包 DP | 容量和物品 | 分割等和子集、零钱兑换 |
| 子序列 DP | 以 i 结尾或前 i 个 | LIS、LCS |
| 区间 DP | `dp[l][r]` 区间答案 | 戳气球、石子合并 |
| 树形 DP | 子树返回状态 | 二叉树打家劫舍 |

## Java 21 示例：零钱兑换

```java
import java.util.*;

class Solution {
    public int coinChange(int[] coins, int amount) {
        int inf = amount + 1;
        int[] dp = new int[amount + 1];
        Arrays.fill(dp, inf);
        dp[0] = 0;
        for (int a = 1; a <= amount; a++) {
            for (int coin : coins) {
                if (a >= coin) {
                    dp[a] = Math.min(dp[a], dp[a - coin] + 1);
                }
            }
        }
        return dp[amount] == inf ? -1 : dp[amount];
    }
}
```

## Python 3 示例：零钱兑换

```python
class Solution:
    def coinChange(self, coins: list[int], amount: int) -> int:
        inf = amount + 1
        dp = [inf] * (amount + 1)
        dp[0] = 0
        for a in range(1, amount + 1):
            for coin in coins:
                if a >= coin:
                    dp[a] = min(dp[a], dp[a - coin] + 1)
        return -1 if dp[amount] == inf else dp[amount]
```

## 验收

- 能先写中文状态定义，再写代码。
- 能解释为什么遍历顺序不能随意交换。

> **重点：** 状态定义决定转移方程，不能先写循环再猜 `dp` 含义。
>
> **难点：** 背包问题中一维压缩后的遍历方向非常关键。
>
> **易错：** 把“子数组”DP 和“子序列”DP 混淆。
