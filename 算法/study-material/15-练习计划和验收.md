## 阶段验收

### 初级

- 能完成简单题。
- 能讲清楚哈希、双指针、栈队列和二分。
- 能写出 Java 21 / Python 3 基本模板。

### 中级

- 能稳定完成中等题。
- 能独立处理树、图、回溯、DP 基础题。
- 能给出复杂度和边界测试。

### 高级

- 能处理陌生变形题。
- 能证明贪心或 DP 转移的正确性。
- 能在面试中边写边解释取舍。

## 自测题

1. 为什么两数之和适合哈希表？
2. 滑动窗口为什么不适合所有子数组和问题？
3. 二分答案的前提是什么？
4. DFS 和 BFS 分别适合什么场景？
5. DP 的状态定义为什么比代码更重要？
6. LRU 为什么需要哈希表和双向链表组合？

## 最终验收

- 任选 10 道中等题，限时 30 分钟一题。
- 每题提交前写 3 个边界用例。
- 每题口述题解不超过 3 分钟。
- 错题 48 小时内重做一次，一周后再重做一次。

> **重点：** 刷题数量只是输入，复盘质量才决定能力增长。
>
> **易错：** 只记录 AC 代码，不记录为什么第一次没想到。
