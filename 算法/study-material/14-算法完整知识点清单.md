# 算法学习资料：算法完整知识点清单

[返回索引](../算法学习资料.md)

## 基础和术语

- 输入规模、约束条件、边界条件。
- 时间复杂度、空间复杂度、均摊复杂度。
- 递归、迭代、分治、剪枝。
- 稳定性、原地算法、比较排序下界。

## 数据结构

- 数组、字符串、矩阵。
- 链表：单链表、双链表、快慢指针、哨兵节点。
- 栈、队列、双端队列。
- 哈希表、哈希集合、计数器。
- 堆、优先队列。
- 树：二叉树、二叉搜索树、Trie、线段树、树状数组。
- 图：邻接表、邻接矩阵、无向图、有向图、加权图。
- 并查集。

## 核心算法范式

- 双指针、滑动窗口。
- 前缀和、差分、扫描线。
- 排序、二分查找、二分答案。
- 单调栈、单调队列。
- DFS、BFS、回溯。
- 动态规划：一维、二维、背包、子序列、区间、树形、状态压缩。
- 贪心：区间、跳跃、股票、交换论证。
- 图算法：拓扑排序、最短路、最小生成树入门。
- 位运算、数学、随机化。

## LeetCode 高频题型

- 数组：两数之和、三数之和、合并区间、旋转数组。
- 字符串：最长无重复子串、异位词、KMP 入门、回文。
- 链表：反转、合并、找环、删除倒数第 N 个节点。
- 树：遍历、深度、最近公共祖先、序列化。
- 图：岛屿、课程表、腐烂橘子、最短路径。
- DP：爬楼梯、打家劫舍、零钱兑换、最长递增子序列、最长公共子序列。
- 设计题：LRU、最小栈、数据流中位数、随机集合。

## Java 21 必会点

- `HashMap`、`HashSet`、`ArrayDeque`、`PriorityQueue`。
- `Arrays.sort`、`Comparator`、`List.of`。
- `record` 可用于简单不可变结构，但 LeetCode 类定义通常沿用题目给定类。
- `long` 处理溢出。
- 避免使用同步旧类 `Stack`，优先 `ArrayDeque`。

## Python 3 必会点

- `dict`、`set`、`Counter`、`defaultdict`。
- `deque`、`heapq`、`bisect`。
- `list.sort(key=...)`。
- 递归深度和全局变量作用域。
- 大整数天然支持，但不能忽略时间复杂度。

## 常见错误清单

- 不看数据范围，直接写暴力。
- 哈希题把同一个元素重复使用。
- 二分边界不一致导致死循环。
- 滑动窗口遇到负数场景仍强行套模板。
- 回溯答案没有复制路径。
- DP 状态定义含糊。
- 图搜索忘记 visited。
- Java `int` 溢出。
- Python 递归爆栈。

## 验收

- 能完成 80 道精选题，其中简单题 25 道、中等题 50 道、困难题 5 道。
- 每类题至少能讲出一个模板、两个变形和一个易错点。
- 能用 Java 21 和 Python 3 各写出 10 道核心题。
