# 算法学习路线图

这份路线图面向 Java 后端、通用研发和算法面试准备，代码示例以 Java 21 和 Python 3 为主。目标不是背题，而是建立“题目特征 -> 算法模型 -> 复杂度 -> 可证明正确的实现”的能力。

## 阶段 1：复杂度、数据结构和编码基本功

目标：能看懂题目规模，判断应该用 `O(n)`、`O(n log n)` 还是 `O(n^2)`，并能在 Java 21 / Python 3 中熟练操作数组、字符串、哈希表、栈、队列、堆。

需要掌握：

- 时间复杂度、空间复杂度、均摊复杂度
- 数组、字符串、链表、哈希表、栈、队列、堆
- Java 21：`ArrayList`、`HashMap`、`ArrayDeque`、`PriorityQueue`、`record`
- Python 3：`list`、`dict`、`set`、`deque`、`heapq`、`dataclass`

练习：

- LeetCode 1、20、21、49、53、88、206、704

验收：

- 能在不查资料的情况下写出二分查找、反转链表、括号匹配、堆的基本用法。
- 能解释为什么 `HashMap` 平均是 `O(1)`，但最坏情况不一定是 `O(1)`。

> **重点：** 算法学习第一关是复杂度判断。题目给 `n = 10^5` 通常不能用 `O(n^2)`。
>
> **易错：** 只背模板，不看约束条件。正确做法是先读数据规模，再决定算法级别。

## 阶段 2：数组、字符串、双指针和滑动窗口

目标：掌握线性扫描类问题，能用指针表示状态边界，减少不必要的嵌套循环。

需要掌握：

- 快慢指针、左右指针、同向双指针
- 固定窗口、可变窗口
- 原地修改数组、去重、合并、反转
- 子数组、子串、子序列的区别

练习：

- LeetCode 3、11、15、26、27、76、209、283、438

验收：

- 能解释窗口什么时候扩张，什么时候收缩。
- 能区分“最长满足条件”和“最短满足条件”的窗口更新时机。

## 阶段 3：哈希表、前缀和、差分和计数模型

目标：能把“是否出现过”“出现次数”“区间和”“批量区间修改”转化为结构化状态。

需要掌握：

- 哈希去重、哈希计数、索引映射
- 前缀和、二维前缀和
- 差分数组、扫描线入门
- 取模前缀和、异或前缀

练习：

- LeetCode 1、49、128、238、303、304、325、560、974、1094

验收：

- 能推导 `sum(i, j) = prefix[j + 1] - prefix[i]`。
- 能说明为什么“和为 K 的子数组”要统计历史前缀和次数。

## 阶段 4：排序、二分、栈队列、堆和单调结构

目标：掌握“有序性”和“局部最优候选”的维护方式。

需要掌握：

- 二分答案、二分边界
- 快排、归并排序、堆排序的思想
- 单调栈、单调队列
- Top K、数据流中位数、区间最大值

练习：

- LeetCode 33、34、69、74、84、155、215、239、295、739、875

验收：

- 能写出不会死循环的二分查找。
- 能解释单调栈中元素被弹出时代表什么。

## 阶段 5：链表、树、图、DFS 和 BFS

目标：掌握指针结构和关系结构的遍历、状态记录与边界处理。

需要掌握：

- 链表反转、合并、快慢指针找环
- 二叉树前中后序、层序遍历、递归返回值设计
- 图的邻接表、DFS、BFS
- visited、距离、父节点、连通分量

练习：

- LeetCode 102、104、141、142、199、200、207、236、297、994

验收：

- 能把递归函数的返回值说清楚。
- 能说明 BFS 为什么天然适合无权最短路。

## 阶段 6：回溯、动态规划和贪心

目标：能处理“选择序列”“最优子结构”“局部选择”等高频面试难点。

需要掌握：

- 回溯三要素：路径、选择列表、结束条件
- DP 状态定义、转移方程、初始化、遍历顺序
- 一维 DP、二维 DP、背包、区间 DP、树形 DP 入门
- 贪心选择性质和反例验证

练习：

- LeetCode 39、46、51、55、70、121、122、198、300、322、416、435、1143

验收：

- 能先写出状态定义，再写转移，不把 DP 写成凭感觉填表。
- 能说出一个贪心策略为什么正确，或者给出反例推翻错误贪心。

## 阶段 7：高级专题和面试整合

目标：能根据题目特征快速归类，完成中等题稳定 AC，并能讲清楚取舍。

需要掌握：

- 并查集、拓扑排序、最短路入门
- 位运算、数学、组合计数
- Trie、线段树、树状数组入门
- LRU、LFU、随机化集合等设计题

练习：

- LeetCode 146、208、210、232、235、399、460、547、684、721、743

验收：

- 能在 30 分钟内完成常见中等题。
- 能复盘失败题目属于：读题错误、模型错误、边界错误、实现错误还是复杂度错误。

## 推荐学习节奏

| 周期 | 学习重点 | 产出 |
| --- | --- | --- |
| 第 1 周 | 复杂度、数组、字符串、哈希表 | 每天 3 道简单题，整理错因 |
| 第 2 周 | 双指针、滑动窗口、前缀和、二分 | 每天 2 道中等题，写模板 |
| 第 3 周 | 栈队列、堆、链表、树 | 完成 20 道结构题 |
| 第 4 周 | DFS、BFS、回溯、图 | 完成岛屿、排列、拓扑类题 |
| 第 5 周 | 动态规划、贪心 | 每类题写状态定义和证明 |
| 第 6 周 | 设计题、高级结构、模拟面试 | 限时训练和口述题解 |

## 最终能力清单

- 能根据数据规模估算复杂度上限。
- 能识别题目对应的算法模型。
- 能用 Java 21 和 Python 3 写出清晰实现。
- 能说明正确性、复杂度和边界条件。
- 能维护自己的错题本和模板库。
