# Shell学习资料：命令速查

[返回索引](../Shell学习资料.md)

## 脚本头

```bash
#!/usr/bin/env bash
set -Eeuo pipefail
```

## 参数和变量

```bash
echo "$#"
printf '%s\n' "$@"
: "${DB_URL:?missing DB_URL}"
port=${PORT:-8080}
```

## 条件

```bash
[[ -f "$file" ]]
[[ -d "$dir" ]]
[[ -n "$s" ]]
[[ "$a" == "$b" ]]
[[ "$line" =~ ^ERROR ]]
```

## 循环

```bash
for arg in "$@"; do
  echo "$arg"
done

while IFS= read -r line; do
  echo "$line"
done < file
```

## 重定向

```bash
cmd >out.log
cmd 2>err.log
cmd >all.log 2>&1
cmd 2>&1 | tee all.log
```

## find/xargs

```bash
find "$dir" -type f -name '*.log' -print0 | xargs -0 grep -H -- 'ERROR'
find "$dir" -type f -mtime +7 -exec gzip -- {} +
```

## 调试

```bash
bash -n script.sh
bash -x script.sh
shellcheck script.sh
```

## trap

```bash
tmpdir=$(mktemp -d)
trap 'rm -rf -- "$tmpdir"' EXIT
```

## 高风险命令前检查

```bash
[[ -n "$target" ]] || exit 2
target=$(realpath -m -- "$target")
[[ "$target" == /allowed/root/* ]] || exit 2
rm -rf -- "$target"
```

## 重点

- 变量默认加双引号。
- 转发参数用 `"$@"`。
- 文件批处理用 `find -print0`。
- 删除前先 dry-run 和路径校验。
