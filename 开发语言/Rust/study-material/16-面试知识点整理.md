# Rust学习资料：面试知识点整理

[返回索引](../Rust学习资料.md)

这份资料用于 Rust 面试前复习。面试问题按类别拆分在 `面试知识点/` 目录中，覆盖基础语法、所有权、Trait、Cargo、并发、异步、unsafe、生产实践和系统设计。

## 使用方式

先读 [17-Rust完整知识点清单.md](17-Rust完整知识点清单.md)，确认自己是否有明显短板。再按类别刷题，每道题要求做到：直接回答、给出最小代码、说明适用边界和常见陷阱。

## 面试复习标准

- 能解释概念背后的内存或运行时机制。
- 能写出最小 Rust 代码验证答案。
- 能说明什么时候不用 Rust 或不用某个高级特性。
- 能把编译器错误翻译成设计问题。
- 能结合生产场景回答：如何测试、如何排障、如何发布、如何保证依赖安全。

## 高频追问方向

- `String` 和 `&str` 的区别。
- move、Copy、Clone 的区别。
- 生命周期标注到底做什么。
- `impl Trait`、泛型、`dyn Trait` 的区别。
- `Option`、`Result`、panic 的边界。
- `Rc`、`Arc`、`RefCell`、`Mutex` 的适用场景。
- `Send`、`Sync` 如何影响并发和 async。
- Cargo resolver、features 和 MSRV。
- unsafe 的安全不变量和 FFI 风险。
- 借用检查器、Trait coherence、Cargo resolver、Future 状态机、Pin、原子内存序、Miri、profiling 和生产故障复盘。
