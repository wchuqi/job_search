# Java设计模式学习资料：面试知识点整理

[返回索引](../Java设计模式学习资料.md)

这份资料用于面试前复习。面试问题已按知识点拆分到 `面试知识点/` 目录。

## 使用方式

- 先看基础概念和原则，确认能解释“为什么用模式”。
- 再按创建型、结构型、行为型复习代码和对比。
- 最后看框架源码和场景题，练习把模式落到真实项目。

## 面试前速记

- 创建型：封装创建。
- 结构型：组织对象关系。
- 行为型：分配职责和调度流程。
- 选择模式先看变化点，再看耦合方向，最后看扩展成本。
