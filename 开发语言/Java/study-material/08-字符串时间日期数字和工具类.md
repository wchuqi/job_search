# Java 学习资料：字符串、时间日期、数字和工具类

[返回索引](../Java学习资料.md)

## 学习目标

完成本章后，你应该能：

- 正确处理 String、StringBuilder、正则和格式化。
- 使用 java.time 处理日期、时间、时区和时间段。
- 理解 BigDecimal 的适用场景和常见坑。
- 熟悉 Objects、Arrays、Collections、UUID 等常用工具。

## 理论导读：标准库是在替你处理容易出错的日常细节

字符串、时间、数字和工具类看似零散，其实共同点是“日常、高频、容易因为细节出错”。字符串涉及不可变性、编码、正则转义和拼接成本；时间涉及时区、日期和瞬时时间点的差异；金额涉及二进制浮点无法精确表示十进制小数；工具类则把空值判断、数组处理、集合包装、随机数生成这些常见操作标准化。

`String` 不可变可以想象成每次修改都会得到一张新纸，原来的纸不会被涂改。这让字符串可以安全共享，也让字符串常量池和哈希缓存成为可能，但大量循环拼接时也可能创建许多中间对象。`StringBuilder` 则像一块可反复写入的草稿板，适合在一个线程里逐步拼出最终字符串。

时间 API 的关键是区分“时间线上的点”和“人类日历上的表示”。`Instant` 是全世界统一的时间点，适合存储和跨系统传输；`LocalDateTime` 没有时区，只是墙上日历和时钟显示的年月日时分秒；`ZonedDateTime` 才把本地时间和地区规则绑定起来。金额计算同理，要用 `BigDecimal` 明确十进制语义，而不是把二进制浮点误当成精确金额。

## 一、String

`String` 不可变：

```java
String s = "hello";
String upper = s.toUpperCase();
```

原始 `s` 不会改变。

常用方法：

```java
s.isBlank();
s.strip();
s.contains("he");
s.startsWith("h");
s.substring(1, 3);
s.replace("h", "H");
```

> **重点：** 字符串内容比较使用 `equals`，不要用 `==`。

## 二、StringBuilder

循环拼接字符串：

```java
StringBuilder builder = new StringBuilder();
for (String item : items) {
    builder.append(item).append(",");
}
String result = builder.toString();
```

> **易错：** 在大量循环中使用 `+` 拼接可能产生不必要对象，优先使用 `StringBuilder` 或 `StringJoiner`。

## 三、格式化

```java
String message = "name=%s, age=%d".formatted("Alice", 18);
```

`String.format`：

```java
String text = String.format("price=%.2f", 12.345);
```

文本块：

```java
String json = """
        {
          "name": "Alice",
          "age": 18
        }
        """;
```

## 四、正则

```java
Pattern pattern = Pattern.compile("\\d+");
Matcher matcher = pattern.matcher("abc123");
if (matcher.find()) {
    System.out.println(matcher.group());
}
```

> **易错：** Java 字符串中反斜杠需要转义，所以正则 `\d` 要写成 `"\\d"`。

## 五、java.time

常用类型：

| 类型 | 含义 |
| --- | --- |
| `Instant` | 时间线上的瞬时点 |
| `LocalDate` | 无时区日期 |
| `LocalTime` | 无时区时间 |
| `LocalDateTime` | 无时区日期时间 |
| `ZonedDateTime` | 带时区日期时间 |
| `Duration` | 时间量，秒/纳秒 |
| `Period` | 日期量，年/月/日 |

示例：

```java
LocalDate today = LocalDate.now();
LocalDate nextWeek = today.plusWeeks(1);

Instant now = Instant.now();
ZonedDateTime shanghai = now.atZone(ZoneId.of("Asia/Shanghai"));
```

> **重点：** 存储跨系统时间通常使用 `Instant`，展示给用户时再转换为时区。

## 六、日期格式化

```java
DateTimeFormatter formatter = DateTimeFormatter.ofPattern("yyyy-MM-dd HH:mm:ss");
LocalDateTime time = LocalDateTime.parse("2026-06-06 10:00:00", formatter);
```

ISO 格式：

```java
Instant instant = Instant.parse("2026-06-06T02:00:00Z");
```

> **易错：** `YYYY` 是 week-based-year，普通年份应使用 `yyyy`。

## 七、BigDecimal

金额计算使用 BigDecimal：

```java
BigDecimal price = new BigDecimal("12.30");
BigDecimal count = BigDecimal.valueOf(3);
BigDecimal total = price.multiply(count);
```

不要这样：

```java
new BigDecimal(0.1); // 不推荐
```

比较：

```java
new BigDecimal("1.0").equals(new BigDecimal("1.00")); // false
new BigDecimal("1.0").compareTo(new BigDecimal("1.00")); // 0
```

> **重点：** 金额用字符串或 `valueOf` 创建 BigDecimal，比较大小用 `compareTo`。

## 八、Objects

```java
Objects.requireNonNull(name, "name");
Objects.equals(a, b);
Objects.hash(id, name);
```

## 九、Arrays 和 Collections

```java
Arrays.sort(array);
Arrays.asList("a", "b");
Collections.unmodifiableList(list);
Collections.emptyList();
Collections.singletonList("only");
```

> **易错：** `Arrays.asList()` 返回固定大小列表，不能 add/remove。

## 十、UUID 和随机数

```java
UUID id = UUID.randomUUID();
```

普通随机：

```java
Random random = new Random();
```

并发或高性能：

```java
ThreadLocalRandom.current().nextInt(100);
```

安全随机：

```java
SecureRandom secureRandom = new SecureRandom();
```

> **重点：** 密码、token、验证码等安全场景应使用 `SecureRandom`。
