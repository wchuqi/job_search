# Java 学习资料：面试知识点整理（JDK 21）

[返回索引](../Java学习资料.md)

这份资料用于 Java 面试前复习。问题按知识点拆分到 `面试知识点/` 目录，覆盖 JDK 21 版本下的语言基础、集合、并发、JVM、工程实践、数据库、Web 和现代 Java 特性。

## 使用方式

- 先看问题，自己口述答案，再对照参考答案补齐细节。
- 基础面试优先看 01 到 05。
- 后端岗位重点补 06、07、08、09。
- JDK 21 相关问题重点看 04 和 08。
- 最后用 10 做速记和自测。

> **重点：** Java 面试不是只考语法，通常会追问底层原理、场景选择、线程安全、JVM 排障和项目实践。
