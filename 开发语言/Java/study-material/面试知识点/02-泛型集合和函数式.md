## 一、泛型

### 11. Java 泛型的类型擦除是什么？

**参考答案：**

Java 泛型主要在编译期提供类型检查，运行期大部分泛型信息被擦除。例如 `List<String>` 和 `List<Integer>` 运行期通常都是 `ArrayList`。

> **重点：** 类型擦除解释了为什么不能 `new T()`、不能 `List<int>`。

### 12. `List<? extends Number>` 和 `List<? super Integer>` 有什么区别？

**参考答案：**

`? extends Number` 表示元素类型是 Number 或其子类，适合读取为 Number，不适合写入具体元素。`? super Integer` 表示元素类型是 Integer 的父类，适合写入 Integer，读取时通常只能当 Object。

口诀：Producer Extends, Consumer Super。

### 13. `List<Integer>` 是 `List<Number>` 的子类型吗？

**参考答案：**

不是。否则可以把 Double 放进 `List<Integer>`，破坏类型安全。

## 二、集合

### 14. ArrayList 和 LinkedList 怎么选？

**参考答案：**

ArrayList 基于动态数组，随机访问快，追加性能好。LinkedList 基于链表，适合作为 Deque 使用。普通业务中大多数 List 场景优先 ArrayList。

### 15. HashMap 的 key 为什么要正确实现 equals 和 hashCode？

**参考答案：**

HashMap 使用 hashCode 定位桶，再用 equals 判断 key 是否相等。如果二者不一致，可能导致相同业务 key 找不到或重复存储。

### 16. HashMap 是线程安全吗？

**参考答案：**

不是。并发读写 HashMap 可能产生数据不一致。并发场景应使用 ConcurrentHashMap、不可变 Map 或外部同步。

### 17. ConcurrentHashMap 为什么不能存 null？

**参考答案：**

因为并发环境中 null 会造成歧义：无法区分 key 不存在还是 value 就是 null。禁止 null 可以让并发语义更清晰。

## 三、函数式和 Stream

### 18. Lambda 和匿名内部类有什么区别？

**参考答案：**

Lambda 用于函数式接口，语法更简洁，this 指向外部对象。匿名内部类会生成独立匿名类，this 指向匿名类实例。

### 19. Stream 的中间操作和终端操作有什么区别？

**参考答案：**

中间操作返回 Stream，惰性执行，例如 filter、map。终端操作触发计算并产生结果，例如 toList、collect、forEach、count。

### 20. map 和 flatMap 的区别是什么？

**参考答案：**

map 是一对一转换，flatMap 是一对多后展平。例如把多行文本拆成单词列表时使用 flatMap。

### 21. Optional 应该怎么用？

**参考答案：**

Optional 适合表示可能缺失的返回值，鼓励调用方显式处理缺失。不要滥用于字段、方法参数和集合元素。

> **易错：** `optional.get()` 不是好习惯，除非已经确定有值。
