## 一、基础语法

### 1. JDK、JRE、JVM 有什么区别？

**参考答案：**

JVM 是运行 Java 字节码的虚拟机；JRE 是运行 Java 程序所需环境；JDK 是开发工具包，包含 JRE/JVM、编译器和诊断工具。JDK 21 开发时通常安装 JDK 即可。

> **重点：** 开发需要 JDK，运行字节码靠 JVM。

### 2. Java 为什么能跨平台？

**参考答案：**

Java 源码编译为平台无关的字节码，字节码运行在不同平台的 JVM 上。跨平台依赖 JVM 实现，而不是同一份机器码在所有平台直接运行。

### 3. 基本类型和引用类型有什么区别？

**参考答案：**

基本类型直接表示值，包括 8 种：byte、short、int、long、float、double、char、boolean。引用类型保存对象引用，例如 String、数组、类、接口、record。引用变量可能为 null，基本类型不能为 null。

### 4. `==` 和 `equals` 有什么区别？

**参考答案：**

`==` 对基本类型比较值，对引用类型比较是否指向同一对象。`equals` 是对象内容比较方法，具体语义由类实现决定。

```java
String a = new String("java");
String b = new String("java");
System.out.println(a == b);      // false
System.out.println(a.equals(b)); // true
```

> **易错：** 字符串内容比较不要用 `==`。

### 5. Java 是值传递还是引用传递？

**参考答案：**

Java 只有值传递。基本类型传递值副本；引用类型传递的是引用值的副本。方法内可以通过引用修改对象状态，但不能让调用方变量指向新对象。

> **重点：** “引用的副本”不是引用传递。

## 二、面向对象

### 6. 封装、继承、多态分别是什么？

**参考答案：**

封装是把状态和操作状态的规则放在对象内部；继承是子类复用和扩展父类；多态是同一父类型引用在运行期调用不同子类实现。

### 7. 重载和重写有什么区别？

**参考答案：**

重载发生在同一个类中，方法名相同、参数不同，编译期决定。重写发生在子类覆盖父类方法，运行期动态分派。

### 8. 抽象类和接口怎么选？

**参考答案：**

抽象类适合共享状态和模板逻辑；接口适合定义能力、协议和多实现。Java 支持单继承、多接口实现，所以面向能力扩展优先接口。

### 9. 为什么重写 `equals` 必须重写 `hashCode`？

**参考答案：**

HashMap、HashSet 等哈希集合先用 hashCode 定位桶，再用 equals 判断相等。如果两个对象 equals 为 true，但 hashCode 不一致，会导致集合行为错误。

### 10. 为什么说组合优于继承？

**参考答案：**

继承会让子类强依赖父类实现，父类变化可能破坏子类。组合通过持有依赖对象实现复用，耦合更低，更容易测试和替换。

> **易错：** 继承不是代码复用的默认方案，只有稳定 is-a 关系才适合。
