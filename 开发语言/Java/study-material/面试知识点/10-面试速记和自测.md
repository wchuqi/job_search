## 面试速记清单

- Java 跨平台靠字节码和 JVM。
- Java 只有值传递。
- `==` 比较引用，`equals` 比较语义相等。
- 重写 `equals` 必须重写 `hashCode`。
- 泛型运行期主要靠类型擦除。
- `List<Integer>` 不是 `List<Number>`。
- HashMap 非线程安全。
- Stream 是惰性执行，终端操作触发计算。
- Optional 适合返回值，不适合滥用为字段和参数。
- `volatile` 不保证复合操作原子性。
- 线程池要关注线程数、队列和拒绝策略。
- 虚拟线程适合 IO 密集，不加速 CPU 计算。
- JVM 堆存对象，栈存方法栈帧和局部变量。
- GC 通过可达性分析判断对象是否可回收。
- 反射会破坏封装，并受模块系统限制。
- PreparedStatement 防 SQL 注入。
- Spring 事务常见失效原因是自调用、异常处理、代理边界。
- record 适合不可变数据载体。
- sealed 适合有限子类型。
- 预览特性不应默认用于生产代码。
