# Java 学习资料：最终学习验收

[返回索引](../Java学习资料.md)

## 使用方式

这份清单用于判断你是否系统掌握 JDK 21 Java 开发。不要只回答概念，还要能写代码、解释风险和完成项目。

## 一、语言基础

- 能解释 JDK、JRE、JVM。
- 能编译运行 Java 程序。
- 能解释基本类型和引用类型。
- 能说明 Java 参数传递是值传递。
- 能正确使用 switch 表达式。

## 二、OOP 和类型能力

- 能设计类、接口、抽象类。
- 能解释封装、继承、多态。
- 能实现 equals / hashCode。
- 能解释泛型类型擦除。
- 能使用 record、sealed、pattern matching。

## 三、标准库

- 能根据场景选择集合。
- 能使用 Stream 完成分组和聚合。
- 能正确使用 Optional。
- 能处理字符串、时间、BigDecimal。
- 能读写文件和调用 HTTP API。

## 四、并发和 JVM

- 能解释线程安全问题。
- 能使用 synchronized、volatile、Lock、Atomic。
- 能配置线程池。
- 能使用虚拟线程。
- 能解释 JVM 内存区域。
- 能排查 CPU、内存和死锁问题。

## 五、工程实践

- 能创建 JDK 21 Maven / Gradle 项目。
- 能写 JUnit 5 测试。
- 能使用 Mockito。
- 能配置日志。
- 能使用静态检查和覆盖率工具。
- 能分析依赖树。

## 六、后端开发

- 能使用 JDBC。
- 能解释事务 ACID 和隔离级别。
- 能防止 SQL 注入。
- 能构建 REST API。
- 能使用 Spring DI 和事务。
- 能做参数校验和统一异常处理。

## 七、安全和维护

- 能识别敏感信息泄露风险。
- 能说明反序列化风险。
- 能设计空值策略。
- 能说明 JDK 8 升级 JDK 21 的检查项。
- 能解释预览特性为什么不能随意用于生产。

## 八、实践验收任务

完成 [20-综合练习项目.md](20-综合练习项目.md)，并满足：

- JDK 21。
- 使用 record / sealed / switch pattern 至少一种现代特性。
- 使用虚拟线程完成并发任务。
- 使用 JDBC 或 ORM 持久化。
- 有测试。
- 有日志。
- 有异常处理。
- 有简单性能或线程排查记录。

## 最终标准

如果你能完成综合项目，并能回答 [22-面试知识点整理.md](22-面试知识点整理.md) 中的大部分问题，说明你已经具备较完整的 Java 开发基础。
