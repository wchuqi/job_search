# NumPy学习资料：数组创建、转换、保存和读取

[返回索引](../Numpy学习资料.md)

## 学习目标

- 掌握常见数组创建函数。
- 理解 `array`、`asarray`、`copy` 的区别。
- 会保存和读取 `.npy`、`.npz`、文本数据。

## 理论导读

数组创建函数决定初始 shape、dtype 和是否分配新内存。`np.array` 默认会创建数组，必要时复制；`np.asarray` 会尽量复用已有数组；`np.empty` 只分配内存不初始化，适合马上覆盖写入。

## 常用创建函数

```python
import numpy as np

np.zeros((2, 3))
np.ones((2, 3), dtype=np.float32)
np.empty((100, 100))
np.arange(0, 10, 2)
np.linspace(0, 1, 5)
np.eye(3)
np.full((2, 2), 7)
```

## 保存和读取

```python
a = np.arange(10)
np.save("a.npy", a)
loaded = np.load("a.npy")

np.savez("data.npz", x=a, y=a * 2)
data = np.load("data.npz")
print(data["x"])
```

## 练习

创建一个 100x3 的随机特征矩阵，保存为 `.npy`，再读取并验证 shape 和 dtype。

## 验收

- 能选择合适的创建函数。
- 能解释 `empty` 为什么不能直接读取未赋值内容。
- 能用 `.npy` 保存 NumPy 原生数组。

## 重点

- `.npy` 和 `.npz` 保留 dtype 和 shape，比普通 CSV 更适合 NumPy 数组。

## 难点

- 文本文件便于查看，但类型和精度控制不如二进制格式。

## 易错

> **易错：** 使用 `np.empty` 后忘记初始化就参与计算。
>
> 正确做法：只有在马上覆盖所有元素时才用 `empty`。
