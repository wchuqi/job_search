# NumPy学习资料：缺失值、掩码、异常值和数值稳定性

## 学习目标

- 掌握 `nan`、`inf`、布尔 mask 和掩码处理。
- 能处理异常值和非法计算。
- 理解浮点误差和数值稳定性。

## 理论导读

NumPy 中浮点缺失常用 `np.nan` 表示，但整数数组不能直接存储 `nan`。缺失值处理要结合 dtype、mask 和下游算法。浮点计算存在舍入误差，因此比较时要使用容差。

## 示例

```python
import numpy as np

x = np.array([1.0, np.nan, 3.0, np.inf])

mask = np.isfinite(x)
clean = x[mask]

print(np.nanmean(x))
print(np.isclose(0.1 + 0.2, 0.3))
```

## 常见函数

| 目标 | 函数 |
| --- | --- |
| 判断 nan | `np.isnan` |
| 判断 inf | `np.isinf` |
| 判断有限值 | `np.isfinite` |
| 忽略 nan 聚合 | `nanmean`、`nanstd`、`nanmax` |
| 条件替换 | `np.where` |
| 裁剪异常值 | `np.clip` |
| 容差比较 | `np.isclose`、`np.allclose` |

## 练习

给定一个包含 `nan`、`inf` 和异常大值的传感器数组，完成清洗、裁剪、标准化和统计汇总。

## 验收

- 能解释 `nan` 为什么不等于自身。
- 能对浮点结果做容差比较。
- 能根据业务选择删除、填充或保留缺失值。

## 重点

- 缺失值处理是数据契约的一部分，不是最后随便补一步。

## 难点

- 直接删除异常值可能改变样本分布，填充也可能引入偏差。

## 易错

> **易错：** 对包含 `nan` 的数组直接用 `mean`，结果变成 `nan` 后继续传播。
>
> 正确做法：明确使用 `nanmean` 或先构造有效值 mask。
