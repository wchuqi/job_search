## 理论导读

NumPy 数组要求元素类型统一。创建数组时，NumPy 会根据输入推断 dtype，也可以显式指定。数组的维度用 `shape` 表示，二维数组通常约定为 `(行, 列)`，更高维数组要结合业务语义命名。

## 基础示例

```python
import numpy as np

a = np.array([1, 2, 3], dtype=np.float64)
b = np.array([[1, 2, 3], [4, 5, 6]])

print(a + 10)
print(b.shape)      # (2, 3)
print(b.sum(axis=0))  # 每列求和
print(b.sum(axis=1))  # 每行求和
```

## axis 心智模型

`axis` 表示“沿哪个维度折叠或操作”。对 shape 为 `(2, 3)` 的数组：

- `axis=0` 折叠行维度，结果长度是列数。
- `axis=1` 折叠列维度，结果长度是行数。

## 重点

- 写 NumPy 代码时先确认 shape，再写计算。

## 难点

- axis 的结果形状要通过“哪个维度被消掉”来推导。

## 易错

> **易错：** 不看 shape 就直接相加或聚合。
>
> 正确做法：关键步骤打印或断言 shape，尤其在机器学习特征矩阵中。
