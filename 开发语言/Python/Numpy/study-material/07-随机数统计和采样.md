# NumPy学习资料：随机数、统计和采样

[返回索引](../Numpy学习资料.md)

## 学习目标

- 掌握现代随机数 API。
- 会做基础统计、采样和模拟。
- 理解随机种子和可复现性。

## 理论导读

现代 NumPy 推荐使用 `np.random.default_rng()` 创建随机数生成器，而不是长期依赖全局随机状态。这样更容易在测试、模拟和并行任务中控制随机性。

## 示例

```python
import numpy as np

rng = np.random.default_rng(seed=42)

samples = rng.normal(loc=0, scale=1, size=(1000,))
print(samples.mean(), samples.std())

choices = rng.choice(["A", "B", "C"], size=10, p=[0.2, 0.5, 0.3])
```

## 常见统计

```python
x = np.array([1, 2, 3, 4, 100])
np.mean(x)
np.median(x)
np.percentile(x, [25, 50, 75])
np.std(x, ddof=1)
```

## 练习

模拟 10000 次抛硬币，每次抛 20 次，统计正面次数的均值、方差和 95% 分位数。

## 验收

- 能使用 `default_rng` 生成可复现随机样本。
- 能解释 `ddof=1` 的用途。
- 能用分位数识别异常值。

## 重点

- 随机实验要记录种子、分布、样本大小和参数。

## 难点

- 统计函数默认行为不一定符合业务定义，例如总体标准差和样本标准差不同。

## 易错

> **易错：** 在多个模块里随意设置全局随机种子。
>
> 正确做法：显式传入 `Generator`，让随机性成为可控依赖。
