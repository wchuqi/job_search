## 一、ufunc、统计、线性代数和随机数

### 1. ufunc 是什么？

**参考答案：**

ufunc 是 NumPy 的通用函数，通常对数组逐元素执行，并支持广播、类型处理、`out` 输出参数和 `where` 条件计算。例如 `np.add`、`np.sqrt`、`np.maximum`。

> **重点：** ufunc 是 NumPy 向量化计算的重要基础。

### 2. `*`、`@` 和 `np.dot` 有什么区别？

**参考答案：**

`*` 是逐元素乘法；`@` 是矩阵乘法；`np.dot` 的行为随维度变化，对一维是内积，对二维是矩阵乘法，更高维有特定规则。矩阵代码中推荐明确使用 `@` 或专门的线性代数函数。

> **易错：** 把 `*` 当成矩阵乘法。

### 3. 为什么解线性方程时不推荐 `inv(A) @ b`？

**参考答案：**

显式求逆通常更慢且数值稳定性更差。解方程应使用 `np.linalg.solve(A, b)`；最小二乘问题可用 `np.linalg.lstsq`。

> **重点：** 面试中这是数值计算基础判断点。

### 4. 为什么推荐 `np.random.default_rng`？

**参考答案：**

它创建独立随机数生成器，避免全局随机状态污染，更适合测试、模拟和模块化代码。传入 seed 后可复现。

> **难点：** 随机性应该作为依赖显式传递，而不是散落在全局状态里。
