# FastAPI学习资料：面试知识点整理

[返回索引](../FastApi学习资料.md)

这份资料用于面试前复习。面试问题已按知识点拆分到 `面试知识点/` 目录。

## 使用方式

先读基础概念，再读路由和 Pydantic，然后读依赖注入、安全、数据库，最后读异步、测试、部署和排障。回答时不要只背概念，要结合请求链路、项目结构、错误处理和生产风险说明。

## 高频准备清单

- FastAPI 和 Flask/Django 的差异。
- ASGI、Uvicorn、Starlette、Pydantic 的分工。
- 路由匹配顺序、参数来源、422 错误。
- `response_model` 的实际作用。
- `Depends` 的依赖树、缓存和 `yield` 清理。
- sync/async 边界、阻塞调用风险。
- 数据库 Session 生命周期和事务边界。
- JWT、OAuth2、401/403、CORS。
- `TestClient` 和依赖覆盖。
- worker、连接池、健康检查、日志和可观测性。
