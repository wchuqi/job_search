# Pandas学习资料：时间序列、窗口计算和重采样

[返回索引](../Pandas学习资料.md)

## 学习目标

- 掌握日期解析、时间索引、重采样和滚动窗口。
- 理解时区、频率、缺失时间点。
- 能构造常见时间序列指标。

## 理论导读

时间序列数据的关键是时间字段是否被正确解析、是否带时区、是否存在重复或缺失时间点。Pandas 提供 `to_datetime`、`resample`、`rolling`、`shift`、`diff` 等能力。

## 示例

```python
import pandas as pd

df = pd.DataFrame(
    {
        "created_at": ["2026-01-01", "2026-01-02", "2026-01-03"],
        "amount": [10, 20, 30],
    }
)
df["created_at"] = pd.to_datetime(df["created_at"])
daily = df.set_index("created_at").resample("D")["amount"].sum()
rolling_7d = daily.rolling(window=7, min_periods=1).mean()
```

## 常见函数

| 目标 | 方法 |
| --- | --- |
| 解析日期 | `pd.to_datetime` |
| 设置时间索引 | `set_index` |
| 重采样 | `resample` |
| 滚动窗口 | `rolling` |
| 前后偏移 | `shift` |
| 差分 | `diff` |
| 提取年月日 | `dt.year`、`dt.month` |

## 练习

按天统计销售额，计算 7 日滚动均值、环比增长率和每月销售额。

## 验收

- 能正确解析日期并排序。
- 能使用 resample 做周期汇总。
- 能解释 rolling 的窗口口径。

## 重点

- 时间指标必须明确时间字段、时区、周期和窗口闭合规则。

## 难点

- 自然日、业务日、自然周、ISO 周、时区切换都会影响结果。

## 易错

> **易错：** 日期字段仍是字符串就开始排序和筛选。
>
> 正确做法：先用 `pd.to_datetime` 转换并处理解析失败。
