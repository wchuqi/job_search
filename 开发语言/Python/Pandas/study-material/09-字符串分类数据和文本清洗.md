# Pandas学习资料：字符串、分类数据和文本清洗

## 学习目标

- 掌握 `.str` 字符串方法。
- 理解 category dtype 的用途。
- 会做常见文本清洗和枚举字段标准化。

## 理论导读

业务表中大量字段是字符串：城市、渠道、商品名、手机号、身份证、标签。Pandas 提供向量化 `.str` 方法处理文本。低基数枚举字段可以转换为 category，减少内存并提升部分操作性能。

## 示例

```python
import pandas as pd

df = pd.DataFrame({"city": [" 上海 ", "shanghai", None], "channel": ["APP", "web", "APP"]})

df["city_clean"] = (
    df["city"]
    .astype("string")
    .str.strip()
    .str.lower()
    .replace({"shanghai": "上海"})
)

df["channel"] = df["channel"].astype("category")
```

## 常用字符串方法

| 目标 | 方法 |
| --- | --- |
| 去空格 | `str.strip` |
| 大小写 | `str.lower`、`str.upper` |
| 包含判断 | `str.contains` |
| 切分 | `str.split` |
| 正则提取 | `str.extract` |
| 长度 | `str.len` |

## 练习

清洗用户手机号、城市和渠道字段，统一大小写、去除空格、识别非法手机号。

## 验收

- 能使用 `.str` 做向量化文本处理。
- 能选择是否使用 category。
- 能处理缺失字符串。

## 重点

- 字符串字段通常需要标准化，否则 groupby 会把同一业务值拆成多个类别。

## 难点

- 正则提取要考虑缺失、格式不统一和多语言字符。

## 易错

> **易错：** 对含缺失值的 object 列直接调用复杂字符串逻辑。
>
> 正确做法：使用 string dtype，并明确缺失值处理策略。
