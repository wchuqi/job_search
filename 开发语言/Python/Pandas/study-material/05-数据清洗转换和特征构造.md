# Pandas学习资料：数据清洗、转换和特征构造

[返回索引](../Pandas学习资料.md)

## 学习目标

- 掌握去重、过滤、替换、映射、派生列和重命名。
- 会使用 `assign`、`map`、`where`、`clip`、`pipe`。
- 避免逐行 `apply` 滥用。

## 理论导读

数据清洗的目标是把原始数据转换成满足契约的干净表：字段含义明确、类型正确、缺失和异常处理有口径、主键和粒度清晰。Pandas 提供大量向量化方法，应优先使用列操作。

## 示例

```python
import pandas as pd

df = pd.DataFrame({"amount": [10, -5, 2000], "city": ["sh", "bj", None]})

clean = (
    df.assign(
        amount=lambda x: x["amount"].clip(lower=0, upper=1000),
        city=lambda x: x["city"].fillna("unknown").str.upper(),
    )
)
```

## 常见清洗

| 目标 | 方法 |
| --- | --- |
| 重命名 | `rename` |
| 删除重复 | `drop_duplicates` |
| 条件替换 | `where`、`mask` |
| 值映射 | `map`、`replace` |
| 字符串清洗 | `str.strip`、`str.lower` |
| 派生列 | `assign` |

## 练习

清洗订单表：去重、金额转数字、城市标准化、剔除无效订单、添加订单月份字段。

## 验收

- 能写出链式清洗流程。
- 能说明每一步清洗的业务口径。
- 能避免不必要的逐行循环。

## 重点

- 数据清洗要可解释、可复现、可测试。

## 难点

- 异常值不是技术问题，通常需要业务规则定义。

## 易错

> **易错：** 为每行写 `df.apply(lambda row: ..., axis=1)`。
>
> 正确做法：优先使用向量化列操作、`map`、`where`、`groupby.transform`。
