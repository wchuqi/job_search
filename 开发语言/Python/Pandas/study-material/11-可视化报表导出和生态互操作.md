# Pandas学习资料：可视化、报表导出和生态互操作

[返回索引](../Pandas学习资料.md)

## 学习目标

- 会用 Pandas 做快速可视化和报表导出。
- 理解与 NumPy、Matplotlib、SQL、scikit-learn 的互操作。
- 知道导出时的格式和类型风险。

## 理论导读

Pandas 不是专业可视化库，但适合快速探索。最终报表可能导出为 Excel、CSV、Parquet 或数据库表。与其他库交互时，要注意索引、列名、dtype 和缺失值。

## 示例

```python
import pandas as pd

summary = pd.DataFrame({"month": ["2026-01", "2026-02"], "amount": [100, 150]})
summary.plot(x="month", y="amount", kind="bar")
summary.to_excel("report.xlsx", index=False)
```

## 互操作

| 目标 | 方法 |
| --- | --- |
| 转 NumPy | `to_numpy` |
| 读 SQL | `read_sql` |
| 写 CSV | `to_csv` |
| 写 Excel | `to_excel` |
| 写 Parquet | `to_parquet` |
| 快速画图 | `df.plot` |

## 练习

生成销售月报，输出 Excel，并画出每月销售额柱状图。

## 验收

- 能导出不带索引的报表。
- 能把特征矩阵转给机器学习库。
- 能说明 Excel 和 Parquet 的适用边界。

## 重点

- 对外报表要清理索引、列名、排序和格式。

## 难点

- Excel 面向人读，Parquet 面向机器处理，二者不是同一种交付物。

## 易错

> **易错：** 导出 CSV 时把业务无关 index 一起写出。
>
> 正确做法：多数报表使用 `index=False`，除非索引本身就是业务字段。
