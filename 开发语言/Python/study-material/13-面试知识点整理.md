## 使用方式

先读完整知识点清单，确认自己是否有明显短板。再按类别刷题，每道题要求能做到三点：直接回答、给出例子、说明边界或陷阱。

## 面试复习标准

- 能解释概念，而不是只背结论。
- 能写出最小代码例子验证答案。
- 能说明 Python 3.x 与 Python 2 或旧写法的差异。
- 能说清楚 CPython 相关机制是否依赖具体实现。
- 能结合工程场景回答：如何排障、如何测试、如何保证可复现。

## 高频追问方向

- 为什么可变默认参数危险。
- `is` 和 `==` 的区别。
- 字典为什么查找快，键为什么要可哈希。
- `import` 的查找和缓存机制。
- 迭代器、生成器、装饰器、上下文管理器的底层协议。
- GIL 对线程和 CPU 密集任务的影响。
- 怎样组织一个可测试、可部署的 Python 项目。
- CPython 字节码、frame、引用计数、循环 GC、描述符、元类、类型系统、依赖解析和生产安全。
