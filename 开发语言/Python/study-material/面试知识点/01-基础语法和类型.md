## 一、基础语法和类型

### 1. Python 3.x 和 Python 2 有哪些关键差异？

**参考答案：**

Python 3.x 是当前主线版本，字符串默认是 Unicode，`print` 是函数，`/` 表示真除法，`//` 表示整除，很多迭代 API 返回惰性对象而不是列表。Python 2 已停止维护，现代项目应以 Python 3.x 为准。

```python
print(3 / 2)   # 1.5
print(3 // 2)  # 1
```

> **重点：** 面试中默认讨论 Python 3.x，除非题目明确说 Python 2。
> **易错：** 把 Python 2 的 `print x`、默认编码和整数除法规则带入 Python 3。

### 2. Python 中变量和对象是什么关系？

**参考答案：**

变量名是对象的引用或绑定，不是固定类型的盒子。对象有类型、值和身份，变量名可以重新绑定到不同对象。多个变量名可以绑定同一个可变对象，因此通过任一名字修改对象，其他名字也能看到变化。

```python
a = []
b = a
b.append(1)
print(a)  # [1]
```

> **重点：** 赋值通常不是复制对象，而是建立名字到对象的绑定。
> **难点：** 这个模型会影响参数传递、可变对象、浅拷贝和默认参数。

### 3. `is` 和 `==` 有什么区别？

**参考答案：**

`is` 比较对象身份，即两个名字是否指向同一个对象；`==` 比较值是否相等，具体由对象的 `__eq__` 方法决定。判断 `None` 时使用 `is None`，普通值比较使用 `==`。

```python
a = [1, 2]
b = [1, 2]
print(a == b)  # True
print(a is b)  # False
```

> **重点：** `is` 用于身份判断，`==` 用于值判断。
> **易错：** 因为小整数或字符串驻留导致某些例子里 `is` 看似可用，但这不是值比较规则。

### 4. 可变对象和不可变对象有什么区别？

**参考答案：**

可变对象可以在原对象上修改，例如 `list`、`dict`、`set`；不可变对象创建后值不能原地改变，例如 `int`、`str`、`tuple`。不可变并不等于深层不可变，元组里如果包含列表，列表仍然可以修改。

```python
t = (1, [])
t[1].append(2)
print(t)  # (1, [2])
```

> **重点：** 判断可变性要看对象本身能否原地改变。
> **难点：** 容器不可变和容器内部元素是否可变是两件事。

### 5. 字典为什么查找快？键为什么必须可哈希？

**参考答案：**

字典基于哈希表实现，平均情况下可以接近 O(1) 查找。键必须可哈希，因为字典需要根据键的哈希值定位桶位，同时键在作为字典键期间不应改变影响哈希和相等性的状态。

```python
lookup = {"alice": 90, "bob": 80}
print(lookup["alice"])
```

> **重点：** 字典适合键值映射和快速查找。
> **易错：** 用列表做字典键会报错，因为列表可变且不可哈希。

### 6. 浅拷贝和深拷贝有什么区别？

**参考答案：**

浅拷贝复制外层容器，但内部对象仍然共享；深拷贝递归复制内部对象。嵌套可变结构中，浅拷贝容易出现内部数据互相影响。

```python
import copy

a = [[1], [2]]
b = copy.copy(a)
c = copy.deepcopy(a)
a[0].append(99)
print(b)  # [[1, 99], [2]]
print(c)  # [[1], [2]]
```

> **重点：** 嵌套可变对象要特别关注拷贝层级。
> **易错：** 认为 `a[:]` 或 `list(a)` 是深拷贝。

### 7. Python 的真值判断规则是什么？

**参考答案：**

在布尔上下文中，`False`、`None`、数值零、空字符串、空容器通常为假，非空对象通常为真。自定义对象可以通过 `__bool__` 或 `__len__` 控制真值判断。

```python
if []:
    print("will not run")

if [0]:
    print("will run")
```

> **重点：** 空和假值可以简化条件判断。
> **易错：** 把 `None`、空列表、0 混为一种业务含义；需要区分时应显式判断。
