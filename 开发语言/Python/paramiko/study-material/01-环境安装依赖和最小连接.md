# Paramiko学习资料：环境安装、依赖和最小连接

[返回索引](../Paramiko学习资料.md)

## 学习目标

- 安装 Paramiko 并验证版本。
- 完成最小 SSH 连接和远程命令执行。
- 掌握连接参数和资源关闭。

## 理论导读

Paramiko 依赖底层加密库完成 SSH 协议和加密算法。使用前需要有可访问的 SSH Server、用户名、认证方式和可信 Host Key。脚本中必须显式设置超时，并确保连接最终关闭。

## 安装和验证

```powershell
pip install paramiko
python -c "import paramiko; print(paramiko.__version__)"
```

## 最小示例

```python
import paramiko


def run(host: str, username: str, command: str) -> tuple[int, str, str]:
    client = paramiko.SSHClient()
    client.load_system_host_keys()
    try:
        client.connect(hostname=host, username=username, timeout=10, banner_timeout=10, auth_timeout=10)
        stdin, stdout, stderr = client.exec_command(command, timeout=10)
        exit_code = stdout.channel.recv_exit_status()
        return exit_code, stdout.read().decode(), stderr.read().decode()
    finally:
        client.close()
```

## 常见连接参数

| 参数 | 作用 |
| --- | --- |
| `hostname` | 主机名或 IP |
| `port` | SSH 端口，默认 22 |
| `username` | 用户名 |
| `password` | 密码认证 |
| `key_filename` | 私钥文件 |
| `timeout` | TCP 连接超时 |
| `banner_timeout` | SSH banner 超时 |
| `auth_timeout` | 认证超时 |

## 练习

封装一个 `run_command(host, command)` 函数，返回退出码、标准输出、标准错误和耗时。

## 验收

- 能安装并打印 Paramiko 版本。
- 能执行一条远程命令并读取退出码。
- 连接异常时也能关闭 client。

## 重点

- 连接、认证、命令执行都要设置超时。

## 难点

- 本地 shell 环境和远程非交互 shell 环境不同，命令路径和环境变量可能不一致。

## 易错

> **易错：** 只调用 `stdout.read()`，不读取退出码和 stderr。
>
> 正确做法：同时收集 exit code、stdout、stderr，并把失败信息记录清楚。
