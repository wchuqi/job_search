# Paramiko学习资料：面试知识点整理

## 使用方式

先读 SSH 基础和安全模型，再读连接、命令执行、SFTP，然后读跳板机、并发、超时、排障和生产安全。回答时要突出 Host Key、退出码、超时、幂等和凭据安全。

- Paramiko 的用途和边界。

- Host Key 校验和用户认证的区别。

- 日志脱敏和生产审计。
