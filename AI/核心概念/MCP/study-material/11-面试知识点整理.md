# MCP 学习资料：面试知识点整理

[返回索引](../MCP学习资料.md)

这份资料用于 MCP 面试前复习。面试问题已按知识点拆分到 `面试知识点/` 目录。建议先看完整知识点清单，再按薄弱项刷题。

## 使用方式

- AI 应用工程岗：重点看基础概念、服务端能力、工具 Schema 和 Agent 集成。
- 后端/平台岗：重点看协议生命周期、传输、生产治理和版本管理。
- 安全岗：重点看授权、安全、信任边界和高危工具治理。
- 面试前：用每个问题标题自问自答，答不出来再看参考答案。

## 面试答题标准

- 先定义：说明概念是什么。
- 再边界：说明它不是什么。
- 讲机制：协议消息、能力协商、权限链路要清楚。
- 给例子：用具体 server、tool、resource 说明。
- 说风险：安全、版本、观测和生产治理不能漏。
