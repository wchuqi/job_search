## 一、基础概念

### 1. 什么是 AI Agent？

**参考答案：**

AI Agent 是能围绕目标观察环境、进行推理、选择动作、调用工具、更新状态并根据反馈继续执行的 AI 系统。它和普通 LLM 应用的区别是：Agent 不只是生成文本，还能在受控环境中多步行动。

> **重点：** Agent = 模型 + 工具 + 状态 + 控制循环 + 约束。

### 2. Agent 和 Chatbot 有什么区别？

**参考答案：**

Chatbot 主要通过对话生成回答，不一定能调用工具或改变外部状态。Agent 则可以根据目标动态选择工具、执行动作、观察结果并继续下一步。一个客服 Chatbot 可以回答退款规则，客服 Agent 可以查询订单、判断政策、创建工单。

> **易错：** 不是用了 LLM 就是 Agent。

### 3. Agent 和 Workflow 有什么区别？

**参考答案：**

Workflow 的步骤通常固定，适合确定性流程；Agent 的路径可以根据中间结果动态变化。生产系统常常混合二者：固定步骤用 workflow，不确定决策用 Agent。

> **重点：** 能固定就固定，需要动态决策才引入 Agent。
