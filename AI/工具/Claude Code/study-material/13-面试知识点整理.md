# Claude Code 学习资料：面试知识点整理

[返回索引](../Claude Code学习资料.md)

这份资料用于面试前复习。Claude Code 面试不只考“会不会用命令”，更常考 Agentic Coding、上下文工程、权限安全、工具集成、团队治理和真实工程场景。

## 使用方式

- 先读完整知识点清单，确认覆盖范围。
- 再按类别阅读下面的面试题。
- 回答时尽量结合真实开发流程：读代码、改动、测试、审计、回滚。
- 遇到命令细节，说明“具体参数以当前版本帮助为准”，重点讲稳定机制。
