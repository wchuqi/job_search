## 使用方式

- 先读 [14-MyBatis完整知识点清单.md](14-MyBatis完整知识点清单.md)，确认覆盖范围。
- 基础薄弱者先复习 Mapper、动态 SQL、ResultMap。
- 有项目经验者重点准备执行流程、缓存、插件、事务、性能排障和 SQL 安全。
- 回答时按“结论、机制、例子、风险、排查”组织。

## 高频考察点

- MyBatis 和 JDBC、JPA 的区别。
- MapperProxy、MappedStatement、BoundSql、Executor 执行链路。
- `#{}` 和 `${}` 的区别。
- 动态 SQL 标签和 OGNL。
- resultType、resultMap、association、collection。
- 一级缓存、二级缓存。
- 插件拦截器和分页插件原理。
- Spring Boot 集成和事务。
- N+1、慢 SQL、大分页、批量写。
- SQL 注入和生产规范。
- CacheKey、对象折叠、插件代理链、Spring 事务绑定 SqlSession、statementId 级排障。
