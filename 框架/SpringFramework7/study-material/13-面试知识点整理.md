# Spring Framework 7 学习资料：面试知识点整理

[返回索引](../SpringFramework7学习资料.md)

这份资料用于面试前复习。面试问题已按知识点拆分到 `面试知识点/` 目录。

## 使用方式

- 先读完整知识点清单，确认没有大范围遗漏。
- 再按下面顺序刷题：基础概念 -> IoC -> AOP/事务 -> Web/数据 -> 生产场景。
- 每道题先自己口述 1 分钟，再对照参考答案补齐机制、边界和易错点。

## 高频考点

- Spring 和 Spring Boot 的区别。
- BeanFactory 和 ApplicationContext 的区别。
- Bean 生命周期和 BeanPostProcessor。
- 自动装配多候选 Bean 如何选择。
- AOP 代理失效场景。
- `@Transactional` 传播行为和回滚规则。
- Spring MVC 请求处理链路。
- Filter 和 Interceptor 的区别。
- 统一异常处理、参数校验、消息转换。
- 常见启动失败和接口异常排障。
