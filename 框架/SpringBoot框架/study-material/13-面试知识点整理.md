# Spring Boot 学习资料：面试知识点整理

[返回索引](../SpringBoot学习资料.md)

这份资料用于面试前复习。面试问题已按知识点拆分到 `面试知识点/` 目录。

## 使用方式

- 先读 [14-SpringBoot完整知识点清单.md](14-SpringBoot完整知识点清单.md)，确认覆盖范围。
- 对基础薄弱者，先复习自动配置、配置体系、Web、事务。
- 对有项目经验者，重点准备生产运维、排障、迁移和场景题。
- 回答面试题时按“结论、机制、例子、风险”组织，不要只背定义。

## 面试答题框架

1. 先给直接结论。
2. 再解释底层机制或执行链路。
3. 给项目中的具体例子。
4. 补充风险、边界和排查方式。

## 高频考察点

- Spring Boot 和 Spring Framework 的关系。
- 自动配置条件和覆盖规则。
- `@SpringBootApplication` 的组成。
- 配置优先级和 Profile。
- Controller 参数绑定、校验、异常处理。
- 事务传播、回滚和失效。
- Spring Security Filter Chain。
- Actuator 安全暴露和可观测性。
- 测试切片和 Testcontainers。
- Boot 3 到 Boot 4 的迁移风险。
- 请求链路、事务代理、配置覆盖、连接池耗尽等深度场景表达。
