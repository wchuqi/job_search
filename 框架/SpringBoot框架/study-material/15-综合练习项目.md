# Spring Boot 4 学习资料：综合练习项目

[返回索引](../SpringBoot4学习资料.md)

## 项目目标

实现一个 Spring Boot 4 订单服务，覆盖自动配置、配置绑定、Web、数据访问、事务、Flyway、Actuator、测试、容器化和生产排障。

## 功能范围

- 创建订单。
- 查询订单。
- 取消订单。
- 订单审计日志。
- PostgreSQL 持久化。
- Flyway 迁移。
- 统一错误响应。
- Actuator health/metrics/prometheus。
- Testcontainers 集成测试。
- 自定义 Starter：审计客户端。

## 推荐结构

```text
boot-order-service/
```

## 必做任务

1. 使用 Boot 4.0.6。
2. 引入 Web MVC、Actuator、JDBC、Validation、Flyway、Testcontainers。
3. 使用 `@ConfigurationProperties` 管理业务配置。
4. 使用 Flyway 创建表。
5. Service 使用 `@Transactional`。
6. Controller 使用 DTO，不直接返回 Entity。
7. 统一错误响应包含 code、message、traceId。
8. 自定义 HealthIndicator 检查支付网关。
9. 自定义 Meter 记录订单创建数。
10. 使用 Testcontainers 做 PostgreSQL 集成测试。
11. 构建 OCI 镜像。
12. 写 5 份故障复盘。

## 验收标准

- `mvn test` 通过。
- 应用可通过环境变量切换数据库。
- `/actuator/health` 区分 readiness/liveness。
- `/actuator/prometheus` 有自定义指标。
- 自动配置条件报告能解释关键 Bean 来源。
- 测试包含 Web 切片和真实数据库集成测试。
- 容器镜像不包含生产密钥。
