# Spring Boot 4 学习资料：深度实验手册和能力验收

[返回索引](../SpringBoot4学习资料.md)

## 学习目标

用可运行实验验证 Spring Boot 核心机制。每个实验都记录：目的、代码、现象、解释、易错、验收。

## 实验 1：启动流程和事件

- 写 ApplicationListener 监听启动事件。
- 写 ApplicationRunner 打印参数。
- 切换 web-application-type。

验收：能说明每个事件大致发生在 refresh 前还是后。

## 实验 2：自动配置和条件报告

- 写自定义 Starter。
- 使用 class、property、missing bean 条件。
- 打开 conditions 报告。

验收：能解释每个自动配置 match/not match 原因。

## 实验 3：配置加载和 Binder

- 使用 application.yml、application-dev.yml、环境变量、命令行参数。
- 使用 `spring.config.import`。
- 写带校验的 `@ConfigurationProperties`。

验收：能说出最终值来自哪个配置源。

## 实验 4：Web 容器和 MVC

- 修改端口、context-path。
- 注册 Filter。
- 自定义 WebServerFactoryCustomizer。
- 写 REST API 和统一错误。

验收：能解释请求从 Web 容器到 Controller 的链路。

## 实验 5：数据库和事务

- Flyway 建表。
- JdbcTemplate 写入。
- Service 事务。
- Testcontainers 验证 PostgreSQL。

验收：能证明异常后事务回滚，迁移脚本在测试中执行。

## 实验 6：Actuator 和指标

- 开启 health、metrics、prometheus、conditions。
- 自定义 HealthIndicator。
- 自定义 Counter/Timer。
- 动态调整日志级别。

验收：能设计 readiness/liveness 和 Prometheus 指标。

## 实验 7：打包部署

- `mvn package`。
- `java -jar` 运行。
- `spring-boot:build-image` 构建镜像。
- 环境变量注入配置。

验收：镜像可运行，配置不写死，健康检查可用。

## 实验 8：故障复盘

至少复盘：

- 自动配置不生效。
- 配置覆盖错误。
- 端口占用。
- 健康检查误杀。
- 测试上下文慢。
- 依赖版本冲突。

验收：每份复盘包含现象、复现、根因、修复、回归测试、预防。

## 最终能力验收

- 能讲清 `SpringApplication.run()`。
- 能讲清自动配置导入和条件评估。
- 能定位配置来源。
- 能写企业 Starter。
- 能设计 Actuator 暴露和健康分组。
- 能写切片测试和 Testcontainers 集成测试。
- 能构建容器镜像。
- 能评估 Boot 4 升级风险。
