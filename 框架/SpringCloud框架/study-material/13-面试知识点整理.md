这份资料用于面试前复习。面试问题按知识点拆分到 `面试知识点/` 目录。

## 使用方式

- 先读 [14-SpringCloud完整知识点清单.md](14-SpringCloud完整知识点清单.md)，确认覆盖范围。
- 基础薄弱者先复习版本、服务发现、Feign、Gateway。
- 有项目经验者重点准备熔断限流、配置中心、消息可靠性、Kubernetes、生产排障。
- 回答时按“结论、机制、项目例子、风险、排查”组织。

## 高频考察点

- Spring Cloud 和 Spring Boot 的关系。
- Release Train 和 BOM 的作用。
- 服务注册发现和客户端负载均衡。
- OpenFeign 的调用链路、超时、错误处理。
- Gateway 的 Route、Predicate、Filter。
- Config Server、`spring.config.import`、动态刷新边界。
- CircuitBreaker、Retry、Bulkhead、RateLimiter 的区别。
- Stream 消费组、幂等、DLQ、outbox。
- Kubernetes 与 Spring Cloud 的边界。
- 微服务安全、Token 传播、Actuator 保护。
- 跨服务 trace、指标和生产排障。
- 实例筛选、Feign 代理、Gateway 过滤链、Config 加载时机、消息 ack、级联故障治理。
