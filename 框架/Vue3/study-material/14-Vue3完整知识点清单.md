# Vue3学习资料：完整知识点清单

[返回索引](../Vue3学习资料.md)

这份清单用于检查 Vue3 学习覆盖面。若准备面试，要求至少能对每个一级知识域讲出用途、机制、典型场景和常见坑。

## 1. 基础和环境

- Vue3 定位、渐进式框架、声明式渲染、组件化。
- Vue2 到 Vue3：Proxy、组合式 API、Fragments、Teleport、Suspense、编译优化。
- Vite、create-vue、SFC、`script setup`、TypeScript。
- 项目结构：views、components、composables、stores、api、types、utils。
- 环境变量、路径别名、静态资源、构建产物、部署 base。

## 2. 模板和指令

- 插值、表达式限制、HTML 转义和 XSS 风险。
- `v-bind`、`v-on`、`v-if`、`v-show`、`v-for`、`v-model`。
- 事件修饰符、按键修饰符、表单修饰符。
- class/style 对象和数组绑定。
- `key`、列表 diff、排序插入删除的状态稳定性。

## 3. 响应式系统

- `ref`、`reactive`、`readonly`、`shallowRef`、`shallowReactive`。
- `computed` 缓存、getter/setter computed。
- `watch` source、deep、immediate、flush、cleanup。
- `watchEffect`、`watchPostEffect`、`watchSyncEffect`。
- `toRef`、`toRefs`、`unref`、`isRef`、`markRaw`、`toRaw`。
- Proxy track/trigger、effect、scheduler、批处理。
- 响应式失效：解构、整体替换、浅响应式、外部对象、数组和 Map/Set。

## 4. 组件系统

- SFC、组件注册、局部组件、异步组件。
- props 类型、默认值、校验、只读边界。
- emits、事件命名、`v-model` 参数和多个 `v-model`。
- slots、具名插槽、作用域插槽、默认内容。
- attrs 透传、`inheritAttrs`、组件根节点。
- provide/inject、依赖注入边界。
- `defineExpose`、模板 ref、动态组件。

## 5. 生命周期和组合式 API

- `setup` 执行时机。
- `onMounted`、`onUpdated`、`onUnmounted`。
- `onActivated`、`onDeactivated`、KeepAlive。
- `onErrorCaptured` 错误捕获边界。
- composable 设计、实例私有状态、共享状态、清理副作用。

## 6. Router

- 路由模式：hash、history。
- 静态路由、动态路由、嵌套路由、命名路由。
- params、query、props 解耦。
- 导航守卫：全局、路由独享、组件内。
- 路由懒加载、滚动行为、导航失败。
- 权限路由、动态菜单、404 顺序、无限重定向。
- 参数变化组件复用和 `onBeforeRouteUpdate`。

## 7. Pinia 和状态架构

- option store 与 setup store。
- state、getters、actions、storeToRefs。
- 插件、持久化、订阅、重置。
- 状态分类：组件本地、URL、Pinia、服务端缓存。
- 登录态、权限、主题、购物车、刷新恢复。
- 安全：token 存储、退出清理、敏感信息边界。

## 8. 渲染机制和内置组件

- 模板编译、render function、vnode。
- patch、patch flag、block tree、静态提升。
- scheduler、`nextTick`、更新批处理。
- Teleport、KeepAlive、Transition、TransitionGroup、Suspense。
- 异步组件、错误加载、超时和骨架屏。

## 9. 表单、请求和业务状态

- 表单草稿、字段校验、提交状态、防重复提交。
- loading、error、empty、success 状态。
- 请求取消、竞态保护、重试、缓存、乐观更新和回滚。
- API 层封装、拦截器、鉴权刷新。
- 文件上传、进度、失败重试。

## 10. 性能和大型应用

- 首屏优化、拆包、懒加载、预加载、依赖分析。
- 大列表：分页、虚拟列表、浅响应式。
- 重复渲染定位、组件拆分、computed 缓存。
- `v-once`、`v-memo`、KeepAlive 边界。
- 图片优化、缓存策略、CDN。
- 内存泄漏：定时器、事件、第三方实例、未取消请求。

## 11. 测试和调试

- Vue DevTools、组件树、事件、Pinia、性能。
- Vitest、Vue Test Utils、Playwright/Cypress。
- mock 请求、路由测试、Pinia 测试。
- ESLint、Prettier、TypeScript、提交检查、CI。
- Source Map、错误监控、日志和埋点。

## 12. SSR、部署和安全

- CSR、SSR、SSG、hydration。
- Nuxt 或 Vite SSR 基础概念。
- hydration mismatch 原因和排查。
- Nginx history fallback、缓存头、gzip/brotli。
- XSS、CSRF、CSP、CORS、Cookie SameSite。
- 前端环境变量不保存密钥。

## 13. 生产排障

- 白屏、资源 404、构建 base 错误。
- 路由刷新 404、权限重定向循环。
- 响应式不更新、DOM 读取旧值、列表状态串位。
- 内存泄漏、页面卡顿、接口重复请求。
- SSR 水合不一致、客户端专属 API 报错。

## 14. 面试能力检查

- 能用 2 分钟讲清 Vue3 更新链路。
- 能解释 `ref`、`reactive`、`computed`、`watch` 的边界。
- 能设计一个中后台的路由和权限架构。
- 能说明 Pinia 状态分类和持久化风险。
- 能拿一个性能问题说出定位过程和指标。

## 15. 深度机制补充清单

- 响应式：`ReactiveEffect`、依赖清理、嵌套 effect、`effectScope`、`customRef`、`triggerRef`、调试钩子。
- 调度：job 去重、pre flush、post flush、同步 flush、`nextTick` 与微任务队列。
- 编译器：parse、transform、codegen、SFC 宏、作用域 CSS、静态提升、事件缓存。
- 渲染优化：patch flag、dynamicChildren、block tree、fragment 稳定性、SSR hydration 提示。
- 组件实例：props 解析、attrs 透传、slots 归一化、expose、错误捕获、异步组件状态机。
- 内置组件：KeepAlive 缓存 key、LRU、activated/deactivated；Teleport 目标移动；Suspense fallback。
- Router：path ranking、matcher、参数解析、导航失败、守卫完整执行顺序、动态路由重置。
- Pinia：store 生命周期、插件扩展、订阅、action hook、SSR 序列化和 XSS 风险。
- 请求状态：取消、竞态、缓存键、失效策略、乐观更新、幂等、错误恢复。
- 性能：构建拆包、依赖体积、长任务、布局抖动、内存泄漏、虚拟列表和浅响应式取舍。
- SSR：跨请求状态污染、hydration mismatch、客户端专属 API、Cookie/CSRF、Nuxt 与纯 Vue SSR 边界。
- TypeScript：`defineProps` 类型推导、泛型组件、插槽类型、emits 类型、store 类型、路由 meta 类型扩展。
- 测试：组件契约、composable 测试、Pinia/Router mock、网络 mock、E2E 稳定性、可访问性断言。
