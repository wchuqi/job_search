# Vue3学习路线图

这份路线图面向已经掌握 HTML、CSS、JavaScript 基础的前端或后端开发者，目标是从能写页面，进阶到能维护中大型 Vue3 应用。

## 阶段 1：基础认知

- 目标：理解 Vue3 解决的问题、项目结构、声明式渲染和组件化思想。
- 需要掌握：Vite 项目、SFC 单文件组件、模板语法、指令、事件、样式作用域、开发调试。
- 例子：用 `create-vue` 创建项目，完成待办事项列表。
- 练习：实现列表新增、完成、删除、筛选。
- 验收：能解释 `template`、`script setup`、`style scoped` 的职责。
- 重点：Vue 不是 DOM 操作工具，而是“状态驱动视图”的 UI 运行时。
- 易错：把 Vue 当 jQuery 用，频繁手写 `querySelector` 修改 DOM。

## 阶段 2：响应式和组件核心

- 目标：掌握 `ref`、`reactive`、`computed`、`watch`、props、emits、slot。
- 需要掌握：响应式依赖收集、组件单向数据流、事件上抛、插槽内容分发。
- 例子：拆分搜索框、列表、详情弹窗、分页组件。
- 练习：写一个可复用 `UserTable`，支持搜索、排序、空状态和加载状态。
- 验收：能判断何时用 `computed`，何时用 `watch`，能避免直接修改 props。
- 重点：状态变化会触发组件重新渲染；组件边界决定维护成本。
- 易错：滥用深度监听、把所有数据塞进一个巨大对象。

## 阶段 3：组合式 API 和工程化

- 目标：用组合式 API 管理复杂逻辑，建立可测试、可复用的业务代码。
- 需要掌握：`setup`、生命周期钩子、composable、异步请求、错误状态、TypeScript、ESLint、Prettier。
- 例子：封装 `useRequest`、`usePagination`、`useFormDirty`。
- 练习：把一个页面中的请求、表单、筛选逻辑拆成 composable。
- 验收：业务组件只负责渲染和编排，复杂逻辑可以单独测试。
- 重点：组合式 API 的核心价值是按“业务能力”组织代码。
- 难点：避免 composable 隐式依赖太多上下文。

## 阶段 4：路由、状态和应用架构

- 目标：构建 SPA 的页面组织、权限控制、跨页面状态和缓存策略。
- 需要掌握：Vue Router 路由匹配、动态路由、导航守卫、懒加载、Pinia、持久化、权限菜单。
- 例子：后台管理系统：登录、用户列表、角色权限、详情页。
- 练习：实现登录后按权限生成菜单，并在刷新后恢复登录态。
- 验收：能说明路由守卫执行顺序和 Pinia 与组件本地状态的边界。
- 重点：路由管理页面位置，Pinia 管理跨页面共享状态。
- 易错：把接口临时数据、表单输入、UI 展开状态全部放入全局 store。

## 阶段 5：运行机制、性能和生产化

- 目标：理解 Vue3 运行时机制，并能定位性能、SSR、水合和生产问题。
- 需要掌握：虚拟 DOM、patch、scheduler、`nextTick`、key、异步组件、KeepAlive、Teleport、Suspense、构建优化、SSR/hydration。
- 例子：优化大列表、首屏拆包、分析重复渲染、处理水合不一致。
- 练习：用性能面板找出一个列表页卡顿原因并修复。
- 验收：能解释为什么错误的 `key` 会导致状态串位，为什么同步读取 DOM 可能拿到旧结果。
- 重点：性能优化先定位瓶颈，再选择缓存、拆分、虚拟列表、懒加载等手段。
- 难点：响应式更新、组件渲染和浏览器绘制不是同一个时刻。

## 阶段 6：源码机制和架构深水区

- 目标：能从源码模型解释 Vue3 的关键机制，并能把机制用于设计、排障和面试追问。
- 需要掌握：`ReactiveEffect`、`EffectScope`、scheduler、SFC 编译、patch flag、block tree、Router matcher、导航解析、Pinia 插件、SSR hydration、类型宏、测试分层。
- 例子：手写最小响应式系统、分析模板编译产物、模拟 keyed diff、实现权限路由决策表。
- 练习：选择一个真实页面，输出响应式依赖图、组件更新路径、路由权限路径、构建拆包报告和性能火焰图。
- 验收：能回答“为什么这样设计”，而不是只回答“怎么使用 API”。
- 重点：源码学习要服务工程判断，避免只背函数名。
- 难点：编译器、运行时、响应式、路由和状态管理是互相影响的系统。

## 推荐学习节奏

| 周期 | 学习重点 | 产出 |
| --- | --- | --- |
| 第 1 周 | 环境、模板、指令、组件基础 | Todo 应用 |
| 第 2 周 | 响应式、组件通信、组合式 API | 可复用表格和表单 |
| 第 3 周 | Router、Pinia、请求和权限 | 小型后台管理系统 |
| 第 4 周 | 测试、性能、部署、SSR 基础 | 带测试和部署脚本的项目 |
| 第 5 周 | 响应式、渲染和编译优化深度解析 | 最小响应式实现和编译产物分析 |
| 第 6 周 | Router、Pinia、请求状态和权限架构 | 权限后台架构说明书 |
| 第 7 周 | SSR、水合、部署、安全和性能排障 | 生产排障剧本和性能报告 |
| 第 8 周以后 | 源码阅读、测试治理、面试追问 | 源码实验手册和面试复盘 |

## 最终能力清单

- 能独立搭建 Vue3 + Vite + TypeScript 项目。
- 能设计清晰的组件边界和状态边界。
- 能熟练使用组合式 API 封装业务逻辑。
- 能使用 Vue Router 和 Pinia 搭建中型 SPA。
- 能写组件测试、组合函数测试和关键 E2E 用例。
- 能排查响应式失效、重复渲染、路由守卫死循环、水合不一致等问题。
- 能回答 Vue3 高频面试题，并能结合项目场景说明取舍。
- 能读懂 Vue3 关键编译产物，知道静态提升、patch flag、block tree 为什么存在。
- 能把“组件状态、本地状态、URL 状态、Pinia 状态、服务端缓存”分层建模。
